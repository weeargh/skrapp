## Prerequisites  <!-- confidence:high ✓ -->

Untuk melihat custom report di Qontak CRM, Anda memerlukan:

• Akun Qontak CRM aktif dengan kredensial login
• Peran pengguna: Admin atau Owner (untuk mengajukan custom report)
• Custom report sudah diajukan sebelumnya melalui menu Report dan telah disetujui
• Akses ke halaman Dashboard untuk melihat tab Custom Reports

Catatan: Custom report hanya akan terlihat setelah Anda mengajukan request dan pihak Qontak telah memproses permintaan tersebut.

## Steps  <!-- confidence:high ✓ -->

1. Login ke akun CRM Qontak Anda menggunakan kredensial Anda
   → Halaman Dashboard Qontak CRM akan terbuka

2. Pada halaman Dashboard, klik tab **Custom Reports**
   → Daftar semua custom report yang Anda ajukan akan ditampilkan

3. Lihat hasil custom report yang sudah diproses
   → Isi custom report akan sesuai dengan kesepakatan dan form yang Anda isi sebelumnya

4. (Opsional) Untuk mengunduh report, klik ikon **Download** pada kolom report yang ingin diunduh
   → File report akan diunduh ke perangkat Anda

## Expected Result  <!-- confidence:high ✓ -->

Setelah mengikuti langkah-langkah di atas, Anda berhasil melihat custom report di tab Custom Reports pada Dashboard. Halaman menampilkan daftar semua custom report yang telah Anda ajukan dengan data yang sudah disesuaikan sesuai permintaan Anda. Anda dapat melihat detail laporan dan mengunduhnya kapan saja melalui tombol Download yang tersedia di setiap baris report.

## Error States  <!-- confidence:medium ~ -->

Jika custom report tidak muncul di tab Custom Reports:

• Pastikan custom report sudah diajukan sebelumnya dan sudah disetujui oleh tim Qontak
• Refresh halaman browser Anda (tekan F5) untuk memastikan data terbaru dimuat
• Periksa apakah peran pengguna Anda adalah Admin atau Owner, karena peran lain mungkin tidak memiliki akses ke Custom Reports
• Hubungi support jika report masih tidak muncul setelah pengajuan

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim support Qontak jika mengalami masalah:

• Custom report tidak muncul di Dashboard meskipun sudah diajukan lebih dari waktu yang ditentukan
• Menerima pesan error saat mengakses tab Custom Reports
• Kesulitan mengunduh file report

Siapkan informasi berikut saat menghubungi support: ID akun Qontak Anda, nama custom report yang bermasalah, kapan Anda mengajukan request, dan screenshot halaman Dashboard Anda.
