## Prerequisites  <!-- confidence:high ✓ -->

Untuk mengunduh data Deals di Qontak CRM, Anda memerlukan:

1. Akun Qontak CRM aktif dengan akses login
2. Akses ke menu Deals (role Admin atau Sales disarankan)
3. Minimal satu Deal yang sudah tersimpan dalam sistem
4. Koneksi internet yang stabil untuk proses pengunduhan
5. Ruang penyimpanan yang cukup di perangkat Anda untuk file yang akan diunduh

## Steps  <!-- confidence:high ✓ -->

1. Login ke akun Qontak CRM Anda.
   → Dashboard utama akan ditampilkan.

2. Klik menu **Deals** di navigasi utama.
   → Halaman daftar Deals akan muncul.

3. Klik tombol **"List View"** di sebelah kanan halaman Deals.
   → Mode tampilan list akan aktif.

4. Centang checkbox pada Deal yang ingin Anda unduh. Untuk mengunduh semua data Deals, centang **"Select all filtered results"**.
   → Deals yang dipilih akan ditandai.

5. Klik tombol **"Download"**.
   → Pop-up pemilihan field akan muncul.

6. Pilih field yang ingin Anda unduh dengan mencentang checkbox. Gunakan pencarian untuk menemukan field dengan cepat. Hapus centang pada field yang tidak diperlukan. Pastikan minimal 2 field tercentang.
   → Data field terpilih akan ditampilkan.

7. Klik tombol **"Download"** pada pop-up.
   → Notifikasi pengunduhan sedang diproses akan muncul.

8. Untuk mengecek progress pengunduhan, buka menu **Properties** lalu pilih tab **Upload/Download**.
   → Status dan progress pengunduhan akan ditampilkan.

## Expected Result  <!-- confidence:high ✓ -->

Setelah proses pengunduhan selesai, file data Deals akan tersimpan di folder download perangkat Anda dalam format spreadsheet (CSV atau Excel). File berisi kolom-kolom field yang Anda pilih sebelumnya. Anda dapat membuka file menggunakan aplikasi spreadsheet seperti Microsoft Excel atau Google Sheets untuk melihat dan menganalisis data Deals Anda.

## Error States  <!-- confidence:medium ~ -->

1. **Minimal 2 field tidak tercentang**: Sistem tidak akan mengizinkan pengunduhan jika kurang dari 2 field dipilih. Solusi: Centang minimal 2 field sebelum klik tombol Download.

2. **Koneksi internet terputus**: Proses pengunduhan dapat terganggu. Solusi: Periksa koneksi internet Anda, tunggu hingga stabil, lalu coba kembali.

3. **File gagal diunduh**: Kemungkinan disebabkan ukuran file terlalu besar atau masalah browser. Solusi: Gunakan browser yang berbeda atau filter data untuk mengurangi jumlah Deal sebelum mengunduh.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim support Qontak jika Anda mengalami:

1. File tidak dapat diunduh meskipun sudah mengikuti semua langkah
2. Notifikasi pengunduhan tidak pernah selesai setelah waktu yang sangat lama
3. Data Deal tidak sesuai dengan yang ditampilkan di layar
4. Error message yang tidak jelas saat mengunduh

Siapkan informasi: screenshot dari halaman yang bermasalah, jumlah Deal yang ingin diunduh, dan waktu kejadian error.
