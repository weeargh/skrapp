## Prerequisites  <!-- confidence:high ✓ -->

• Akun Qontak Mobile CRM yang aktif
• Aplikasi Mobile CRM terinstall di perangkat smartphone
• Akses ke menu Settings dalam aplikasi Mobile CRM
• Koneksi internet untuk konfigurasi awal dan sinkronisasi data
• Peran pengguna dengan akses ke fitur CRM (Contact, Company, Deals, Tasks, Notes, Products)

## Steps  <!-- confidence:high ✓ -->

**Konfigurasi Offline Mode:**
1. Buka aplikasi Mobile CRM dan masuk ke akun Anda.
2. Ketuk menu "Other" di bagian bawah navigasi.
3. Pilih menu "Account" pada bagian Settings.
4. Gulir ke bawah hingga menemukan opsi "Offline mode".
5. Pilih salah satu dari tiga mode: Automatic (beralih otomatis saat tidak ada internet), Always online (tetap online sepanjang waktu), atau Always offline (beroperasi offline dan sinkronisasi manual).
6. Konfirmasikan pilihan Anda. Aplikasi akan menyesuaikan fungsinya berdasarkan pengaturan yang dipilih.

**Mengakses Unsynchronized Data:**
1. Buka aplikasi Mobile CRM.
2. Ketuk menu "Other" lalu pilih tab "Unsynchronized Data".
3. Periksa data pada setiap tab (Deals dan Tasks dapat dikelola saat ini).
4. Saat koneksi internet tersedia, aplikasi secara otomatis menyinkronkan data yang pending.## Expected Result  <!-- confidence:high ✓ -->

• Mode offline dikonfigurasi sesuai preferensi pengguna (Automatic, Always online, atau Always offline).
• Halaman "Unsynchronized Data" menampilkan semua data yang dibuat atau diperbarui saat offline.
• Data yang tidak tersinkronisasi tersimpan aman di aplikasi hingga koneksi internet dipulihkan.
• Saat koneksi internet tersedia, data pada tab Deals dan Tasks menyinkronisasi secara otomatis ke server.
• Notifikasi muncul untuk memberi tahu status sinkronisasi data.

## Error States  <!-- confidence:medium ~ -->

• Data tidak menyinkronisasi otomatis setelah koneksi internet dipulihkan: Verifikasi koneksi internet stabil, tutup dan buka kembali aplikasi Mobile CRM, atau coba sinkronisasi manual melalui halaman Unsynchronized Data.
• Tab Contact dan Company tidak dapat dikelola di halaman Unsynchronized Data: Fitur offline untuk tab ini belum siap; hanya tab Deals dan Tasks yang dapat dikelola untuk saat ini.
• Aplikasi tetap offline meskipun koneksi internet tersedia: Periksa pengaturan mode offline Anda; jika mode Always offline dipilih, sinkronisasi data secara manual.

## Escalation  <!-- confidence:medium ~ -->

Hubungi Qontak support jika:
• Data yang tidak tersinkronisasi hilang atau tidak muncul di halaman Unsynchronized Data setelah koneksi internet dipulihkan.
• Aplikasi crash atau error saat mengakses fitur Offline Mode.
• Pengaturan Offline Mode tidak dapat diubah atau tidak responsif.
• Sinkronisasi data gagal berulang kali meskipun koneksi internet stabil.

Sertakan informasi: screenshot halaman error, nama mode offline yang dipilih, dan daftar perangkat yang terpengaruh.
