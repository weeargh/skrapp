## Prerequisites  <!-- confidence:medium ~ -->

Untuk memahami hasil download Laporan Target By Months By Count di Qontak CRM, Anda membutuhkan:

• Akun Qontak CRM aktif dengan kredensial login
• Peran pengguna: Admin, Supervisor, atau Manager (untuk mengakses laporan target)
• Akses ke menu Report atau Dashboard di Qontak CRM
• Data deals dengan status "won" sudah tersedia di sistem
• Minimal satu target bulanan sudah ditetapkan untuk user atau tim

Catatan: Laporan ini menampilkan perbandingan antara target dan pencapaian count deals per user pembuat deals.

## Steps  <!-- confidence:medium ~ -->


Laporan ini dapat Anda gunakan untuk melihat perbandingan berdasarkan user pembuat deals, antara jumlah Count dalam **stage won** dengan target jumlah Count pada bulan yang anda pilih.
Hasil download **Laporan Target By Months By Count** ketika sudah di download akan menampilkan sheet seperti gambar di bawah ini:

## Escalation  <!-- confidence:medium ~ -->

Hubungi dukungan Qontak jika Anda mengalami:

• Laporan tidak menampilkan data deals yang sudah ada di sistem
• Filter tidak berfungsi saat memilih bulan, waktu, atau user
• File Excel tidak dapat diunduh atau rusak saat proses download
• Nilai Target atau Achieved Count menunjukkan angka yang tidak sesuai

Saat menghubungi support, siapkan: screenshot laporan, user ID, periode laporan yang difilter, dan nama-nama user yang seharusnya muncul di laporan.
