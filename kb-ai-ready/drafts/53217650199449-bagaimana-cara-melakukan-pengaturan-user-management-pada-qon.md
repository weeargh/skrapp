## Prerequisites  <!-- confidence:high ✓ -->

Untuk melakukan pengaturan User Management pada Qontak One, Anda memerlukan:

• Akun Qontak One yang sudah aktif
• Peran Admin (hanya Admin yang memiliki akses ke fungsi User Management)
• Akses ke menu Settings > Account > Users
• Kuota user yang tersedia (jika ingin menambah user baru)
• Plan tier yang mendukung fitur User Management

## Steps  <!-- confidence:high ✓ -->


Fungsi **User Management** pada Qontak One memungkinkan Anda untuk menambahkan, mengubah, dan menghapus pengguna yang sudah ada. Selain itu, Anda juga dapat memantau **User quota**. Harap diperhatikan bahwa hanya peran **Admin** yang memiliki akses ke fungsi ini. Untuk lebih jelasnya, simak penjelasan berikut.

**No.** | **Nama Fitur** | **Penjelasan**  
---|---|---  
1 | Active | Menampilkan jumlah user yang aktif.  
2 | Pending | Menampilkan jumlah user yang _invitation_ -nya masih mengalami _pending_ dari sistem dan belum menyelesaikan proses registrasi pada email.  
3 | Available quota | Menampilkan jumlah sisa kuota user yang masih dapat ditambahkan.  
4 | Invite user | Tombol untuk menambahkan user baru.  
5 | Upgrade quota | Tombol untuk melakukan _upgrade_ kuota.  
6 | All roles | Filter untuk menyaring data _role_ user yang telah dibuat. Dalam hal ini, Anda dapat melakukan filter data dari _role-role_ yang ingin dimunculkan dalam tabel.  
7 | All status | Filter untuk menampilkan status user pada akun. Dalam hal ini, Anda dapat melakukan filter status data dari _role-role_ yang ingin dimunculkan dalam tabel.  
8 | Search user | Untuk mencari data user yang telah terbuat.  
9 | Full name | Kolom yang menampilkan nama panjang user.  
10 | Role | Kolom yang menampilkan peran-peran user pada akun.   
11 | Status | Kolom yang menampilkan status user pada akun.  
12 | Last sign-in | Kolom yang menampilkan tanggal dan waktu terakhir user melakukan _sign-in_ pada akun.  
13 | Actions | Tombol untuk mengakses halaman data user secara _detail_ dan menghapus user.

## Error States  <!-- confidence:medium ~ -->

**Kuota User Habis:**
Jika kuota user Anda telah mencapai batas maksimal, sistem akan menampilkan notifikasi pada halaman Users. Anda dapat:
• Klik tombol Upgrade quota untuk menambah kuota user
• Klik tombol Contact support untuk menghubungi tim Qontak

**Status User Tetap 'Pending':**
Jika status user tidak berubah menjadi 'Active' setelah waktu tunggu, periksa kembali email yang dikirimkan dan pastikan user telah mengaktifkan akun melalui link undangan.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim support Qontak melalui tombol Contact support pada halaman Users jika mengalami:

• Kuota user tidak dapat di-upgrade
• User tidak menerima email undangan setelah resend invitation
• Status user tetap 'Pending' lebih dari 24 jam
• Error saat membuat atau mengedit user

Siapkan informasi berikut saat menghubungi support: nama akun Qontak, user ID yang bermasalah, dan screenshot halaman Users menunjukkan kondisi masalah.
