## Prerequisites  <!-- confidence:high ✓ -->

Untuk melakukan pengaturan User Management pada Qontak One, Anda memerlukan:

• Akun Qontak One yang sudah aktif
• Peran Admin (hanya Admin yang memiliki akses ke fungsi User Management)
• Akses ke menu Settings > Account > Users
• Kuota user yang tersedia (jika ingin menambah user baru)
• Plan tier yang mendukung fitur User Management

## Steps  <!-- confidence:high ✓ -->

**Untuk menambahkan user baru:**

1. Masuk ke akun Qontak One Anda
2. Pilih menu Settings
3. Pilih sub menu Account dan klik tab Users
4. Klik tombol Invite users
5. Isi semua field yang tersedia (nama, email, role, dan pengaturan lainnya)
6. Klik tombol Invite user
7. Sistem akan menampilkan daftar user dengan status 'Pending'
8. Tunggu beberapa saat hingga status berubah menjadi 'Active'

**Untuk edit, delete, atau resend invitation:**

1. Pada halaman Users, temukan user yang ingin diubah
2. Klik tombol Actions pada user tersebut
3. Pilih opsi yang diinginkan (Edit, Delete, atau Resend Invitation)

## Expected Result  <!-- confidence:high ✓ -->

Ketika pengaturan User Management berhasil dilakukan, Anda akan melihat:

• Daftar user baru muncul pada halaman Users dengan status 'Pending'
• Setelah beberapa saat, status user berubah menjadi 'Active'
• Working status user yang baru ditambahkan menunjukkan '-' karena belum ada riwayat sign-in
• User dapat diakses pada halaman Users dengan opsi Actions (Edit, Delete, Resend Invitation) tersedia
• Perubahan data user langsung terupdate pada sistem

## Error States  <!-- confidence:medium ~ -->

**Kuota User Habis:**
Jika kuota user Anda telah mencapai batas maksimal, sistem akan menampilkan notifikasi pada halaman Users. Anda dapat:
• Klik tombol Upgrade quota untuk menambah kuota user
• Klik tombol Contact support untuk menghubungi tim Qontak

**Status User Tetap 'Pending':**
Jika status user tidak berubah menjadi 'Active' setelah waktu tunggu, periksa kembali email yang dikirimkan dan pastikan user telah mengaktifkan akun melalui link undangan.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim support Qontak melalui tombol Contact support pada halaman Users jika mengalami:

• Kuota user tidak dapat di-upgrade
• User tidak menerima email undangan setelah resend invitation
• Status user tetap 'Pending' lebih dari 24 jam
• Error saat membuat atau mengedit user

Siapkan informasi berikut saat menghubungi support: nama akun Qontak, user ID yang bermasalah, dan screenshot halaman Users menunjukkan kondisi masalah.
