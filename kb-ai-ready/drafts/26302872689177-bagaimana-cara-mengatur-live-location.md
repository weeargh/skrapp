## Prerequisites  <!-- confidence:medium ~ -->

Untuk mengatur Live Location di Qontak CRM, Anda memerlukan:

1. Akun Qontak CRM aktif dengan peran Admin atau Manager
2. Akses ke menu Profile Settings di dasbor CRM web
3. Perangkat dengan koneksi internet stabil
4. Izin lokasi perangkat sudah diaktifkan (untuk fitur tracking aktif)

Tidak ada persyaratan plan tier khusus yang tercantum dalam dokumentasi.

## Steps  <!-- confidence:high ✓ -->

Ikuti langkah-langkah berikut sesuai dengan kebutuhan Anda:

**Opsi A: Live Location dengan Fitur Tracking**
1. Klik tanda panah di sebelah kanan username Anda (kanan atas dasbor CRM) → pilih **Profile settings**
2. Klik **Live location** → aktifkan toggle **Live tracking** dan **Multiple live location**
3. Pilih intensitas tracking: Intensive tracking, Standard tracking, atau Battery-saver tracking
4. Klik **Save changes** → pengaturan tersimpan

**Opsi B: Multiple Live Location Tanpa Tracking**
1. Klik tanda panah di sebelah kanan username → pilih **Profile settings**
2. Klik **Live location** → aktifkan **Multiple live location** dan nonaktifkan **Live tracking**
3. Klik **Save changes**

**Opsi C: Single Live Location Tanpa Tracking**
1. Klik tanda panah di sebelah kanan username → pilih **Profile settings**
2. Klik **Live location** → nonaktifkan **Live tracking** dan **Multiple live location**
3. Klik **Save changes**

## Expected Result  <!-- confidence:high ✓ -->

Setelah klik **Save changes**, sistem menampilkan konfirmasi bahwa pengaturan Live Location berhasil disimpan. Pengaturan yang dipilih akan langsung berlaku di akun Qontak CRM Anda:

- Jika memilih Opsi A: Live tracking aktif dengan intensitas sesuai pilihan, dan Multiple live location otomatis aktif (terkunci)
- Jika memilih Opsi B: Multiple live location aktif untuk merekam beberapa kehadiran tanpa tracking otomatis
- Jika memilih Opsi C: Hanya satu lokasi yang dapat direkam per sesi absensi

## Error States  <!-- confidence:high ✓ -->

No common errors documented.

## Escalation  <!-- confidence:medium ~ -->

Jika Anda mengalami masalah saat mengatur Live Location:

1. Verifikasi bahwa Anda memiliki peran Admin atau Manager di akun Qontak CRM
2. Pastikan browser sudah di-refresh atau coba akses dari browser lain
3. Periksa koneksi internet stabil sebelum klik **Save changes**

Jika masalah persisten, hubungi tim support Qontak dengan informasi: nama akun, screenshot halaman Live location settings, dan langkah yang sudah dicoba.
