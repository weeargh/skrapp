## Definition  <!-- confidence:high ✓ -->

Menu Reports adalah fitur di Qontak Omnichannel yang menyediakan laporan terperinci tentang aktivitas percakapan, kinerja agent, broadcast, dan metrik customer satisfaction. Melalui menu Reports, Anda dapat mengakses tujuh jenis laporan berbeda: General, Agent Performance, Broadcast, Contacts, Bot Performance, CSAT, CES, NPS, Call Activity, dan Export. Setiap laporan dirancang untuk memberikan insights spesifik tentang operasional omnichannel Anda dari berbagai perspektif.

## Why It Matters  <!-- confidence:medium ~ -->

Menu Reports membantu Anda memantau kesehatan operasional omnichannel secara komprehensif. Dengan akses ke berbagai jenis laporan, Anda dapat menganalisis performa agent, efektivitas broadcast campaign, tingkat kepuasan pelanggan, dan aktivitas percakapan secara real-time. Insights ini penting untuk mengidentifikasi area perbaikan, mengoptimalkan alokasi sumber daya, dan membuat keputusan berdasarkan data untuk meningkatkan customer experience.

## Key Attributes  <!-- confidence:high ✓ -->

• Tujuh tipe report: General, Agent Performance, Broadcast, Contacts, Bot Performance, CSAT, CES, NPS, Call Activity, dan Export
• Filter Timezone: untuk menyaring data berdasarkan zona waktu tertentu
• Filter Channel: untuk menyaring data berdasarkan channel komunikasi (WhatsApp, Telegram, dll)
• Filter Date: untuk menyaring data berdasarkan rentang tanggal spesifik
• Filter Division: untuk menyaring data berdasarkan divisi organisasi
• Format ekspor: tersedia dalam format CSV untuk kemudahan analisis lebih lanjut
