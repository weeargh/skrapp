## Prerequisites  <!-- confidence:high ✓ -->

Anda perlu memiliki akses ke aplikasi Qontak Chat di smartphone dengan role Agent atau di atasnya. Pastikan Anda sudah login dengan akses email dan password yang terdaftar di Qontak. Status online harus diaktifkan agar dapat menerima dan mengelola chat masuk. Room percakapan yang akan di-handover harus sudah ada di Inbox Anda.

## Steps  <!-- confidence:high ✓ -->

1. Buka aplikasi Qontak Chat dan login menggunakan email serta password yang terdaftar.
2. Pastikan status diatur ke Online. Di Inbox, pilih room percakapan yang akan di-handover.
3. Klik tombol Profil di bagian atas room percakapan.
4. Di kolom Agents, klik ikon Edit untuk memulai penugasan.
5. Pilih satu Agent dari daftar (gunakan kolom Search untuk mencari jika diperlukan).
6. Klik tombol Assign pada pop-up konfirmasi yang muncul.
7. Konfirmasi notifikasi sukses ditampilkan di layar Anda.
8. Di Contact Info, ketuk ikon Silang di sebelah nama Agent lama Anda, lalu pilih Remove untuk menghentikan penugasan Anda.

## Expected Result  <!-- confidence:high ✓ -->

Room percakapan berhasil di-handover ke Agent baru. Notifikasi konfirmasi muncul di layar yang menunjukkan penugasan Agent berhasil dilakukan. Agent baru akan segera tampil dalam Contact Info room percakapan. Status Anda sebagai Agent lama di room tersebut akan dihapus setelah Anda memilih Remove.

## Error States  <!-- confidence:medium ~ -->

Jika tombol Assign tidak responsif, periksa apakah Agent yang dipilih memiliki status Online dan workload yang tersedia. Jika notifikasi gagal muncul, coba refresh aplikasi atau login kembali. Jika ikon Edit tidak terlihat di kolom Agents, pastikan Anda memiliki permission untuk assign Agent (hubungi admin Qontak). Jika Agent tidak muncul dalam daftar, pastikan Agent tersebut sudah terdaftar dalam sistem dan diassign ke channel yang sama.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim support Qontak jika: (1) tombol Edit pada kolom Agents tidak muncul meskipun Anda memiliki role yang sesuai; (2) notifikasi konfirmasi tidak muncul setelah klik Assign; (3) Agent yang ingin di-assign tidak muncul dalam daftar Agents; (4) proses handover gagal berulang kali. Siapkan screenshot layar Inbox, Contact Info, dan Agent list, serta informasi email akun Anda.
