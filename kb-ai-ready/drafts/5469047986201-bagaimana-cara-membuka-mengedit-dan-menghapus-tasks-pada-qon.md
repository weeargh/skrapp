## Prerequisites  <!-- confidence:high ✓ -->

Untuk membuka, mengedit, dan menghapus Tasks pada Qontak versi web, Anda memerlukan:

• Akun Mekari Qontak CRM yang aktif
• Login akses ke www.qontak.com
• Akses ke menu Tasks
• Task sudah tersedia dalam sistem CRM Anda
• Web browser untuk mengakses Qontak CRM

Untuk menghapus Task, Anda memerlukan izin yang sesuai berdasarkan role pengguna di sistem.

## Steps  <!-- confidence:high ✓ -->

1. Masuk ke akun Mekari Qontak CRM Anda dan buka menu **Tasks**. Sistem akan menampilkan daftar semua Task yang telah dibuat.

2. Klik Task yang ingin Anda buka atau lihat detailnya. Halaman detail Task akan terbuka dan menampilkan semua informasi Task termasuk Activity Log.

3. Untuk mengedit Task, klik icon **Pensil** pada Task yang ingin diubah. Formulir edit Task akan terbuka.

4. Edit kolom yang ingin Anda ganti. Catatan: kolom **Reporter** tidak dapat diubah setelah Task dibuat (kolom ini dinonaktifkan). Klik tombol **Edit** untuk menyimpan perubahan. Sistem akan menampilkan notifikasi konfirmasi penyimpanan.

5. Untuk memantau perubahan Task, klik **"Open full screen in a new tab"** lalu pilih tab **Activity log**. Semua aktivitas pada Task akan terlihat.

6. Untuk menghapus Task, klik icon **Tempat Sampah** pada Task yang akan dihapus. Dialog konfirmasi akan muncul.

7. Klik **"Remove"** untuk menghapus Task atau klik **"No"** untuk membatalkan penghapusan. Sistem akan menampilkan notifikasi konfirmasi penghapusan.

## Expected Result  <!-- confidence:high ✓ -->

Setelah berhasil menyelesaikan langkah-langkah:

• Task yang telah diedit akan tersimpan dengan perubahan yang diterapkan
• Notifikasi konfirmasi penyimpanan akan ditampilkan di layar
• Activity Log menampilkan semua perubahan yang dilakukan pada Task
• Jika menghapus Task, data Task akan dihapus dari menu Tasks
• Notifikasi konfirmasi penghapusan akan muncul setelah Task berhasil dihapus

## Error States  <!-- confidence:low ? -->

No common errors documented.

## Escalation  <!-- confidence:medium ~ -->

Jika Anda mengalami masalah saat membuka, mengedit, atau menghapus Task, hubungi tim support Mekari Qontak dengan menyediakan informasi berikut:

• Screenshot dari halaman Tasks atau error message yang muncul
• ID akun Mekari Qontak CRM Anda
• Deskripsi detail tentang aksi yang Anda lakukan sebelum terjadi error
• Role/permission yang Anda miliki dalam sistem
• Nama Task yang mengalami masalah (jika diperlukan)
