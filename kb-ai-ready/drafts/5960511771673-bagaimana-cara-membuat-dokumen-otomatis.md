## Prerequisites  <!-- confidence:high ✓ -->

Untuk membuat dokumen otomatis di Qontak CRM, Anda memerlukan:

• Akses ke menu Deals atau Tiket di Qontak CRM
• Template dokumen yang telah dibuat dan diupload ke sistem CRM
• Semua field properties pada Deals/Tiket yang dijadikan Variable ID dalam template harus sudah dibuat
• Data lengkap pada Deal/Tiket yang akan digunakan untuk generate dokumen (semua field yang menjadi Variable ID harus terisi)
• Template Excel dengan Variable ID yang sesuai dengan field properties di CRM

## Steps  <!-- confidence:high ✓ -->

1. Buka menu Deals di Qontak CRM.
2. Klik nomor Deal yang ingin Anda buat dokumennya. Sistem akan menampilkan detail Deal tersebut.
3. Lengkapi semua field properties pada Deal sesuai dengan Variable ID yang ada di template dokumen, kemudian klik tombol Save. Sistem akan menyimpan data ke dalam CRM.
4. Setelah data tersimpan, klik tombol Create Dokumen. Sistem akan mengarahkan Anda ke halaman pemilihan template.
5. Pilih template dokumen yang diinginkan dari daftar template yang telah diupload. Sistem akan menampilkan preview dan mengecek apakah Variable ID dapat terisi sesuai data CRM.
6. Apabila data sudah sesuai, klik tombol Create/Update. Sistem akan generate dokumen otomatis.
7. Dokumen akan ditampilkan di layar. Klik icon Mata untuk preview dokumen, atau klik tombol PDF untuk mengunduh dokumen dalam format PDF ke perangkat Anda.## Expected Result  <!-- confidence:high ✓ -->

Setelah berhasil membuat dokumen otomatis, sistem akan menampilkan halaman dengan dokumen yang telah terbuat. Dokumen tersebut berisi data dari Deal/Tiket sesuai dengan template yang dipilih dan field properties yang telah diisi di CRM.

Anda akan melihat dua pilihan aksi:
• Icon Mata: untuk melakukan preview dokumen di layar
• Tombol PDF: untuk mengunduh dokumen dalam format PDF ke perangkat

Dokumen siap digunakan sesuai kebutuhan bisnis Anda.

## Error States  <!-- confidence:low ? -->

No common errors documented.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim support Qontak jika mengalami masalah berikut:

• Tombol Create Dokumen tidak muncul atau tidak aktif di detail Deal/Tiket
• Halaman pemilihan template tidak menampilkan template yang telah diupload
• Variable ID tidak terisi otomatis meskipun field properties di Deal sudah lengkap
• Dokumen gagal di-generate atau halaman error saat klik Create/Update
• Tombol PDF atau icon Mata tidak berfungsi

Siapkan informasi: nomor Deal, nama template yang digunakan, dan screenshot halaman error.
