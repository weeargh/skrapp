## Prerequisites  <!-- confidence:medium ~ -->

Anda perlu mengatur Follow-up Templates pada Qontak Omnichannel untuk membuat template balasan pesan otomatis kepada pelanggan. Persyaratan yang diperlukan:

• Akun Qontak Omnichannel aktif dengan akses ke menu Campaign (Broadcast)
• Role/izin sebagai Admin atau user dengan akses ke fitur Templates
• Koneksi internet stabil
• Tidak ada batasan plan tier khusus untuk fitur ini

## Steps  <!-- confidence:high ✓ -->

1. Login ke akun Qontak Omnichannel Anda.
2. Buka menu **Campaign** → pilih **Templates**.
3. Klik tombol **"Create template"** → pilih **Follow-up template**.
4. Isi kolom-kolom berikut:
   - **Template name**: Gunakan huruf kecil tanpa spasi
   - **Languages**: Pilih bahasa template
   - **Enter the text for your message**: Ketik pesan Follow-up Anda
5. Format teks jika diperlukan menggunakan Bold, Italics, atau Strikethrough.
6. Klik tombol **"Next"** untuk melanjutkan.
7. Verifikasi semua data sudah benar.
8. Klik tombol **"Submit"** untuk menyimpan template.

## Expected Result  <!-- confidence:high ✓ -->

Setelah Anda klik tombol **"Submit"**, sistem akan menampilkan pop-up notifikasi yang mengonfirmasi bahwa Follow-up template berhasil dibuat. Template kemudian akan tersimpan dan siap digunakan dalam campaign broadcast Anda. Template akan muncul di daftar **Templates** di menu Campaign (Broadcast) untuk digunakan kembali di kampanye berikutnya.

## Error States  <!-- confidence:high ✓ -->

**Kolom Required Tidak Terisi:**
Jika ada kolom yang masih kosong, sistem secara otomatis menampilkan notifikasi "field is required" pada kolom tersebut. Pastikan Template name, Languages, dan pesan sudah diisi sebelum klik tombol Submit.

**Template Name Menggunakan Huruf Kapital atau Spasi:**
Sistem akan menolak penyimpanan jika nama template mengandung huruf kapital atau spasi. Edit Template name menggunakan huruf kecil dan garis hubung atau garis bawah sebagai pemisah.

## Escalation  <!-- confidence:high ✓ -->

Jika Follow-up Template tidak muncul di room chat dengan pesan **"The conversations has been expired"**, lakukan langkah berikut:

1. Update aplikasi Qontak Chat melalui Play Store atau App Store.
2. Jika masalah berlanjut, lakukan install ulang aplikasi Qontak Chat.
3. Jika kedua langkah di atas tidak menyelesaikan masalah, hubungi **WhatsApp Hotline Qontak** dengan melampirkan: spesifikasi perangkat, screenshot kendala, dan informasi akun Anda.
