## Prerequisites  <!-- confidence:medium ~ -->

Anda memerlukan akses Admin untuk mengelola Data Layout pada Customer Data Platform. Akun Mekari Qontak harus berupa klien baru yang menggunakan versi dari akhir Q4 2025 ke depan. Fitur Customer Data Platform harus sudah diaktifkan pada paket langganan Anda. Anda perlu memiliki akses ke menu Settings dan Customizations di Qontak One. Pastikan sudah membuat Custom Properties sebelumnya jika ingin menambahkan field tambahan ke dalam layout.

## Steps  <!-- confidence:high ✓ -->

1. Masuk ke akun Mekari Qontak Anda dan pilih menu Settings.
2. Klik sub menu Customizations, lalu pilih Customers. Sistem akan menampilkan halaman Customers.
3. Pada halaman Customers, klik tombol Open customer layout. Sistem akan membuka halaman Data Layout.
4. Isikan nama layout yang Anda inginkan di kolom yang tersedia.
5. Sistem akan menampilkan rincian tampilan layout dengan daftar field yang dapat ditambahkan atau dihapus.
6. Atur susunan field dan tentukan visibilitas berdasarkan peran pengguna sesuai kebutuhan tim Anda.
7. Pastikan seluruh data telah benar, lalu klik tombol Save changes. Sistem akan menyimpan konfigurasi layout.

## Expected Result  <!-- confidence:high ✓ -->

Setelah klik tombol Save changes, Data Layout berhasil dibuat dan tersimpan. Layout baru akan tersedia untuk digunakan pada halaman Add Single Customer Page, Customer Details Page, Edit Customer Page, Customer Profile pada navigasi samping Inbox, dan Edit Customer Profile pada navigasi samping Inbox—baik di Website maupun aplikasi Mobile (CRM Mobile dan Chat Mobile). Setiap pengguna berdasarkan peran mereka akan melihat field dan data yang telah Anda atur dalam layout tersebut.

## Error States  <!-- confidence:low ? -->

No common errors documented.

## Escalation  <!-- confidence:medium ~ -->

Jika Anda mengalami kesulitan saat membuat atau menyimpan Data Layout, hubungi tim support Mekari Qontak. Siapkan informasi berikut: ID akun Mekari Qontak Anda, nama layout yang sedang Anda buat, daftar field yang ingin ditampilkan, screenshot halaman tempat error terjadi, dan pesan error yang muncul (jika ada). Tim support akan membantu mengidentifikasi masalah dan memberikan solusi.
