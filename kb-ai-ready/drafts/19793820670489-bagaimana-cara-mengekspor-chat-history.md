## Prerequisites  <!-- confidence:high ✓ -->

Untuk mengekspor chat history di Qontak Omnichannel, Anda memerlukan:

- Akun Qontak Omnichannel yang aktif
- Role Customer Service, Agent, atau Supervisor
- Akses ke dashboard Qontak melalui web browser
- Minimal satu pesan masuk di Inbox
- Izin akses ke menu Report dan fitur Export

## Steps  <!-- confidence:high ✓ -->

1. Masuk ke dashboard Qontak Omnichannel
2. Buka menu **Report** di navigasi utama
3. Pilih **Export** dari submenu
4. Klik **"View export history"** pada bagian **Chat** — Sistem akan menampilkan halaman Export history
5. Klik **"Export"** untuk memulai proses baru — Pop-up dialog akan muncul
6. Pilih **Period** (periode waktu) data yang ingin Anda ekspor
7. Pilih **Channel** yang diinginkan (WhatsApp, Instagram, dll.)
8. Opsional: Pilih **Agent** tertentu untuk memfilter data berdasarkan agent
9. Klik **"Export"** — Sistem akan menampilkan status "sedang diproses"
10. Tunggu notifikasi email dengan lampiran file yang sudah diekspor
11. Klik lampiran pada email untuk mengunduh file data chat history

## Expected Result  <!-- confidence:high ✓ -->

Setelah proses ekspor berhasil, Anda akan menerima email notifikasi ke alamat email akun Qontak. Email berisi lampiran file data chat history yang dapat diunduh. File berisi semua pesan masuk sesuai filter yang dipilih (periode, channel, dan agent). Anda dapat membuka dan menganalisis data tersebut menggunakan aplikasi spreadsheet atau text editor.

## Error States  <!-- confidence:medium ~ -->

- **Tombol Export tidak aktif**: Pastikan minimal satu periode dan channel sudah dipilih sebelum klik Export
- **Email notifikasi tidak diterima**: Periksa folder Spam/Junk di email Anda. Jika tidak ada, periksa kembali alamat email terdaftar di akun Qontak
- **File tidak dapat diunduh**: Pastikan koneksi internet stabil saat mengunduh dari email. Coba unduh kembali atau gunakan browser berbeda
- **Proses ekspor memakan waktu lama**: Jika data sangat besar, proses bisa berlangsung beberapa menit. Jangan tutup browser atau logout selama proses

## Escalation  <!-- confidence:medium ~ -->

Jika Anda mengalami masalah yang tidak dapat diselesaikan:

- Hubungi Qontak Support melalui dashboard
- Sertakan informasi berikut: ID akun Qontak, screenshot error, periode dan channel yang dicoba untuk diekspor, waktu ekspor dilakukan, dan browser yang digunakan
- Jika email notifikasi tidak diterima dalam 1 jam, hubungi support dengan nomor referensi dari halaman Export history
