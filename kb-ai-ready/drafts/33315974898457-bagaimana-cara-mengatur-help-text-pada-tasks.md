## Prerequisites  <!-- confidence:high ✓ -->

Untuk mengatur Help Text pada Tasks di Mekari Qontak CRM, Anda memerlukan:

• Akun Mekari Qontak CRM yang aktif
• Role Admin atau Owner
• Akses ke menu Properties
• Tasks sudah tersedia dalam sistem CRM Anda
• Web browser untuk mengakses Qontak CRM

Prerequisite ini memastikan Anda memiliki izin yang diperlukan untuk membuat atau mengedit property Tasks.

## Steps  <!-- confidence:high ✓ -->

1. Masuk ke akun Mekari Qontak CRM Anda dan pilih menu **Properties**. Sistem akan menampilkan halaman pengaturan property.

2. Klik tab **Task**. Anda akan melihat daftar property yang ada untuk Tasks.

3. Untuk membuat property baru dengan Help Text, klik tombol **Create New property**. Halaman formulir property baru akan terbuka.

4. Centang toggle **Add help text** dan masukkan panduan atau informasi (maksimal 135 karakter). Sistem akan menampilkan counter karakter. Klik tombol **Save**. Konfirmasi simpan akan ditampilkan.

5. Untuk menambahkan Help Text ke property yang sudah ada, buka tab **Task**, lalu klik tombol **Edit** pada property tersebut. Centang toggle **Add help text**, masukkan teks, dan klik **Save**.

6. Help Text akan muncul di bawah kolom saat Anda membuat atau mengedit Tasks.

## Expected Result  <!-- confidence:high ✓ -->

Setelah mengatur Help Text pada Tasks, Anda akan melihat:

• Pesan konfirmasi "Help text berhasil disimpan" atau notifikasi sukses di layar
• Property baru atau yang diperbarui muncul dalam daftar tab **Task** di menu Properties
• Saat membuat atau mengedit Tasks, Help Text muncul di bawah kolom property yang sesuai sebagai panduan untuk pengguna
• Teks panduan tersebut akan membantu tim Anda memahami cara mengisi data Tasks dengan benar

## Error States  <!-- confidence:medium ~ -->

Kondisi error yang mungkin terjadi:

• **Teks Help Text melebihi 135 karakter**: Sistem akan menolak penyimpanan dan menampilkan pesan "Teks Help Text terlalu panjang". Kurangi jumlah karakter hingga sesuai batas maksimal.

• **Toggle "Add help text" tidak aktif saat membuat property**: Pastikan Anda melihat checkbox bertanda centang sebelum memasukkan teks.

• **Help Text tidak muncul saat membuat Tasks**: Pastikan Anda telah mengklik tombol **Save** dan refresh halaman Tasks untuk memuat perubahan terbaru.

Jika error terus berlanjut meskipun langkah di atas sudah dilakukan, hubungi tim support Qontak.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim support Mekari Qontak jika:

• Help Text tidak muncul setelah disimpan meskipun sudah refresh halaman
• Fitur **Add help text** tidak tersedia atau tidak dapat diakses di menu Properties
• Terjadi error saat menyimpan Help Text setelah mengikuti semua langkah
• Akun Anda tidak memiliki akses ke menu Properties meskipun memiliki role Admin/Owner

Saat menghubungi support, sertakan:
- ID akun Mekari Qontak CRM Anda
- Nama property yang bermasalah
- Screenshot halaman Properties saat terjadi error
- Uraian singkat masalah yang dialami
