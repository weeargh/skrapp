## Prerequisites  <!-- confidence:high ✓ -->

Untuk menggunakan fitur Live Location pada Mekari Qontak CRM Mobile, Anda memerlukan:

1. Akun Qontak CRM aktif dengan peran Sales atau tim yang ditugaskan melakukan kunjungan klien
2. Aplikasi Qontak CRM Mobile sudah terinstal pada smartphone Anda
3. Fitur Live Location sudah diaktifkan oleh Admin atau Manager melalui Profile Settings di website Qontak CRM
4. Izin lokasi (Location Permission) sudah diberikan ke aplikasi Qontak CRM Mobile
5. Koneksi internet yang stabil pada perangkat Anda
6. Pengaturan lokasi perangkat diatur ke 'Always' agar Live Location tetap berfungsi

## Steps  <!-- confidence:high ✓ -->

1. Buka aplikasi **Qontak CRM Mobile** dan masuk dengan akun Anda
2. Ketuk tab **'Other'** di bagian bawah layar
3. Pilih menu **'Live location'**
4. Baca notifikasi ketentuan penggunaan Live Location yang muncul, lalu ketuk **'Ok, got it'**
5. Ketuk tombol **'Start live location'** untuk mengaktifkan pelacakan lokasi
6. Baca notifikasi pemberitahuan yang muncul dan ketuk **'OK'** untuk mengonfirmasi
7. Ketuk tombol **'Start live location'** sekali lagi untuk memulai pelacakan aktif
8. Opsional: Tambahkan foto dan catatan (Note) selama Live Location aktif
9. Ketuk **'See all'** pada halaman detail Live Location untuk melihat riwayat kehadiran Anda

## Expected Result  <!-- confidence:high ✓ -->

Setelah mengikuti langkah-langkah di atas, Live Location Anda akan aktif dan sistem Qontak CRM akan mulai melacak posisi geografis Anda secara berkala. Manager atau Supervisor tim Anda dapat melihat lokasi real-time melalui fitur **Live GPS Report**. Pelacakan akan terus berjalan selama aplikasi Qontak CRM Mobile aktif di background perangkat Anda. Notifikasi sistem akan muncul mengonfirmasi bahwa Live Location telah dimulai dan koneksi internet Anda stabil.

## Error States  <!-- confidence:medium ~ -->

1. **Lokasi tidak terlacak**: Pastikan izin lokasi pada perangkat Anda diatur ke 'Always' dan bukan 'Only while using the app'. Periksa pengaturan lokasi di Settings perangkat.
2. **Koneksi internet putus**: Live Location memerlukan koneksi internet yang stabil. Jika terputus, pelacakan akan terhenti. Pastikan Anda terhubung ke WiFi atau data seluler yang kuat.
3. **Fitur Live Location tidak muncul**: Hubungi Manager atau Admin Anda untuk memastikan Live Location sudah diaktifkan di Profile Settings website Qontak CRM.
4. **Aplikasi berhenti di background**: Pastikan aplikasi Qontak CRM Mobile tidak ditutup paksa atau dihapus dari background apps oleh sistem perangkat.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim support Qontak jika Anda mengalami:

1. Live Location tetap tidak aktif meskipun sudah mengikuti semua langkah
2. Lokasi yang ditampilkan di Live GPS Report tidak akurat atau tidak update
3. Fitur Live Location hilang dari menu 'Other' di aplikasi mobile
4. Notifikasi error berulang saat mencoba mengaktifkan Live Location

Sediakan informasi berikut saat menghubungi support: nama akun Anda, perangkat yang digunakan, versi aplikasi Qontak CRM Mobile, screenshot error (jika ada), dan waktu masalah terjadi.
