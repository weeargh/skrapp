## Prerequisites  <!-- confidence:high ✓ -->

Untuk menggunakan fitur Live Location pada Mekari Qontak CRM Mobile, Anda memerlukan:

1. Akun Qontak CRM aktif dengan peran Sales atau tim yang ditugaskan melakukan kunjungan klien
2. Aplikasi Qontak CRM Mobile sudah terinstal pada smartphone Anda
3. Fitur Live Location sudah diaktifkan oleh Admin atau Manager melalui Profile Settings di website Qontak CRM
4. Izin lokasi (Location Permission) sudah diberikan ke aplikasi Qontak CRM Mobile
5. Koneksi internet yang stabil pada perangkat Anda
6. Pengaturan lokasi perangkat diatur ke 'Always' agar Live Location tetap berfungsi

## Steps  <!-- confidence:high ✓ -->


Saat ini pada Mekari Qontak CRM terdapat fitur **Live Location** yang akan membantu peran Manajer atau Supervisor dalam melacak secara berkala tim yang memiliki tugas untuk melakukan kunjungan rutin ke klien. Biasanya, fitur ini digunakan oleh klien dengan kebutuhan penjualan.

Sementara para tim Sales melakukan kunjungan, peran Manajer atau Supervisor dari tim Sales dapat melacaknya melalui **Live GPS Report**(Laporan GPS Langsung), sebagai bukti kunjungan serta penggantian biaya transportasi (jika diperlukan). Fitur ini dapat di _setting_ melalui Qontak CRM [Website](https://help-center.qontak.com/hc/id/articles/26302872689177-Bagaimana-Cara-Mengatur-Live-Location). Selanjutnya pada langkah-langkah di bawah ini, Anda akan mempelajari cara menggunakan fitur Live location pada CRM Mobile.

Informasi di atas menyatakan bahwa:  
1. Lokasi Anda akan dilacak oleh sistem kami setelah Anda klik **“Start live location”** dan pelacakan lokasi akan berhenti setelah Anda klik **“End live location”**.  
2. Fitur ini akan terus mengakses lokasi Anda selama aplikasi ini masih **aktif** pada background seluler Anda.  
3. Anda dapat selalu memulai kembali **Live location** ketika Anda secara tidak sengaja mengakhiri **Live location** tersebut.
  4. Selanjutnya, klik **“Start live location”** untuk mengaktifkan.  
![3.png](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F41559033784985)
  5. Setelah **Live location** aktif, maka akan muncul notifikasi berikut. Kemudian, klik **“OK”**.  
![4.png](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F41559023874713)

Informasi di atas menyatakan:  
- Tetap atur lokasi Anda ke **‘Always’** untuk selalu mengaktifkan fitur **Live location**.  
- Pelacakan akan **berhenti** ketika Anda klik **“End live location”**.  
- Pastikan Anda memiliki sambungan internet yang stabil.
  6. Untuk memulai pelacakan, klik kembali tombol **“Start live location”**.  
![5.png](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F41559023876633)

Di sini, Anda dapat secara opsional menambahkan **Photo** dan **Note** selama **Live location** aktif atau sedang dalam kondisi **check-in**.
  7. Lalu Anda akan diarahkan ke halaman detail **Live location**. Klik **“See all”** untuk melihat catatan kehadiran Anda.

  8. Selanjutnya, Anda dapat melihat rincian catatan kehadiran Anda melalui **Live Location log** berikut. Klik tombol ini untuk melihat detail setiap perpindahan lokasi.

Demikian cara mengelola fitur Live location pada CRM Mobile.

## Error States  <!-- confidence:medium ~ -->

1. **Lokasi tidak terlacak**: Pastikan izin lokasi pada perangkat Anda diatur ke 'Always' dan bukan 'Only while using the app'. Periksa pengaturan lokasi di Settings perangkat.
2. **Koneksi internet putus**: Live Location memerlukan koneksi internet yang stabil. Jika terputus, pelacakan akan terhenti. Pastikan Anda terhubung ke WiFi atau data seluler yang kuat.
3. **Fitur Live Location tidak muncul**: Hubungi Manager atau Admin Anda untuk memastikan Live Location sudah diaktifkan di Profile Settings website Qontak CRM.
4. **Aplikasi berhenti di background**: Pastikan aplikasi Qontak CRM Mobile tidak ditutup paksa atau dihapus dari background apps oleh sistem perangkat.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim support Qontak jika Anda mengalami:

1. Live Location tetap tidak aktif meskipun sudah mengikuti semua langkah
2. Lokasi yang ditampilkan di Live GPS Report tidak akurat atau tidak update
3. Fitur Live Location hilang dari menu 'Other' di aplikasi mobile
4. Notifikasi error berulang saat mencoba mengaktifkan Live Location

Sediakan informasi berikut saat menghubungi support: nama akun Anda, perangkat yang digunakan, versi aplikasi Qontak CRM Mobile, screenshot error (jika ada), dan waktu masalah terjadi.
