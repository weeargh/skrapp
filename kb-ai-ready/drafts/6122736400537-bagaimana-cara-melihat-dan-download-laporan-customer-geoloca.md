## Prerequisites  <!-- confidence:high ✓ -->

Untuk melihat dan mengunduh laporan Customer Geolocation Map di Qontak CRM, Anda membutuhkan:

• Akun Qontak CRM aktif dengan kredensial login
• Peran pengguna: Admin, Owner, atau Supervisor
• Akses ke menu Dashboard di Qontak CRM
• Report Customer Geolocation Map sudah ditambahkan ke Dashboard Anda sebelumnya
• Data customer dengan informasi geolokasi (lokasi, latitude, longitude) sudah tersimpan dalam sistem

Catatan: Penambahan report ke Dashboard hanya berlaku untuk pengguna individual yang melakukan tindakan ini.

## Steps  <!-- confidence:high ✓ -->

1. Login ke akun Qontak CRM Anda menggunakan kredensial Anda.
   → Halaman Dashboard utama akan terbuka.

2. Pastikan report Customer Geolocation Map sudah ditampilkan di Dashboard Anda.
   → Jika belum ada, buka menu Report Template, cari Customer Geolocation Map, lalu klik "Add to Dashboard".

3. Pada laporan Customer Geolocation Map di Dashboard, tentukan filter yang ingin Anda lihat (opsional).
   → Data akan tersaring sesuai filter pilihan Anda.

4. Klik ikon garis tiga (⋯) yang berada di sudut laporan.
   → Menu dropdown akan muncul menampilkan opsi unduhan.

5. Pilih "Download XLS" dari menu dropdown.
   → File Excel akan diunduh ke perangkat Anda.

## Expected Result  <!-- confidence:high ✓ -->

Setelah mengunduh laporan Customer Geolocation Map, file Excel akan berisi data semua customer di semua pipeline dengan kolom berikut:

• Type, Nama user, Title, Email, Website
• Place name (lokasi), Nomor telepon
• Latitude (Garis lintang), Longitude (Garis bujur)
• Jumlah min deal size, Jumlah max deal size, Jumlah rata-rata deal size
• Owner, Created at

File dapat langsung dibuka dan dianalisis menggunakan aplikasi spreadsheet seperti Microsoft Excel atau Google Sheets.

## Error States  <!-- confidence:medium ~ -->

No common errors documented.

## Escalation  <!-- confidence:medium ~ -->

Jika Anda mengalami masalah saat mengunduh laporan Customer Geolocation Map, hubungi tim support Qontak dengan informasi berikut:

• Screenshot dari halaman Dashboard menunjukkan laporan Customer Geolocation Map
• Email akun Qontak CRM Anda
• Nama organisasi/company
• Waktu terjadinya masalah
• Pesan error atau deskripsi masalah yang dialami

Jika data customer tidak muncul di laporan, verifikasi bahwa data customer memiliki informasi lokasi, latitude, dan longitude yang lengkap di profil customer.
