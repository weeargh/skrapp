## Prerequisites  <!-- confidence:high ✓ -->

• Lisensi IP Bizphone yang aktif untuk masuk dan menggunakan aplikasi
• Untuk Windows: Windows 7 SP1 atau lebih baru, .NET Framework 4.5.2 atau lebih baru, Visual Studio 2013 dan 2015 32-bit redistributable libraries
• Untuk Mac OS: Tidak ada persyaratan sistem khusus
• Koneksi internet (opsional untuk Windows—installer dapat mengunduh library yang diperlukan secara otomatis)

## Steps  <!-- confidence:high ✓ -->

**Windows:**
1. Klik tautan unduhan Windows
2. Ekstrak file .zip
3. Klik kanan installer → pilih "Run as Administrator"
4. Ikuti langkah installer, klik "Next" di setiap tahap
5. (Opsional) Isi Nama Pengguna dan Organisasi
6. Klik "Install"
7. Klik "Finish" untuk menyelesaikan

**Mac OS:**
1. Klik tautan unduhan Mac OS
2. Ekstrak file .zip
3. Pindahkan Bizphone App ke folder Applications (opsional)
4. Buka aplikasi Bizphone

## Expected Result  <!-- confidence:high ✓ -->

• Aplikasi Bizphone Desktop berhasil diunduh dan diinstal
• Aplikasi diluncurkan secara otomatis (Windows) atau siap digunakan (Mac OS)
• Anda dapat membuka aplikasi dan masuk dengan lisensi IP Bizphone aktif Anda
• Softphone siap untuk melakukan dan menerima panggilan melalui komputer

## Error States  <!-- confidence:low ? -->

No common errors documented.

## Escalation  <!-- confidence:medium ~ -->

Jika Anda mengalami masalah saat mengunduh atau menginstal:
• Pastikan lisensi IP Bizphone Anda aktif
• Verifikasi persyaratan sistem Anda terpenuhi
• Untuk Windows: Pastikan .NET Framework 4.5.2+ dan Visual Studio redistributable libraries terinstal
• Hubungi tim dukungan Qontak dengan informasi: sistem operasi, pesan error, dan status lisensi Bizphone Anda
