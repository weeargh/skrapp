## Prerequisites  <!-- confidence:high ✓ -->

Untuk mengatur notifikasi Live Location Reminder pada Qontak CRM Mobile App, Anda memerlukan:

1. Akun Qontak CRM aktif dengan akses Manager atau Admin
2. Qontak CRM Mobile App sudah terinstal di perangkat smartphone
3. Live Location feature sudah dikonfigurasi di Qontak CRM Web Dashboard (lihat artikel terkait: Bagaimana Cara Mengatur Live Location)
4. Izin lokasi perangkat diaktifkan dengan pengaturan 'Always'
5. Koneksi internet stabil

Pastikan timezone perusahaan Anda sudah dikonfigurasi di halaman settings Properties sebelum mengatur notifikasi reminder.

## Steps  <!-- confidence:high ✓ -->


**No.** | **Nama Kolom/Tombol** | **Deskripsi**  
---|---|---  
1 | Timezone | Menginfokan timezone yang aktif pada suatu company. Setting timezone dapat dilakukan di properties -> click 'settings' page.  
2 | Check-in | Aktifkan toggle button Check-in apabila Anda mengaktifkan live location reminder saat absen masuk.

## Escalation  <!-- confidence:medium ~ -->

Hubungi Qontak Support jika mengalami masalah berikut:

1. Notifikasi reminder tidak muncul meskipun sudah diaktifkan:
   - Sertakan screenshot pengaturan Live Location Reminder
   - Konfirmasi bahwa izin notifikasi aplikasi diaktifkan di pengaturan perangkat
   - Berikan informasi timezone yang dikonfigurasi di perusahaan Anda

2. Tidak dapat mengakses menu Live Location Reminder:
   - Pastikan Qontak CRM Mobile App sudah diperbarui ke versi terbaru
   - Sertakan ID akun Qontak CRM Anda
   - Konfirmasi peran akun Anda (Manager/Admin)

3. Waktu notifikasi tidak sesuai dengan timezone perusahaan:
   - Sertakan screenshot pengaturan timezone di halaman Properties
   - Konfirmasi timezone perangkat Anda
