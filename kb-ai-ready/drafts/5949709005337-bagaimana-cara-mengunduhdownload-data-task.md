## Prerequisites  <!-- confidence:high ✓ -->

Untuk mengunduh data Task pada Mekari Qontak CRM, Anda memerlukan:

• Akun Mekari Qontak CRM yang aktif
• Login access ke www.qontak.com
• Akses ke menu Task
• Role Admin atau Owner (untuk mengakses semua fitur download)
• Web browser yang kompatibel
• Data Task sudah tersedia dalam sistem CRM Anda

## Steps  <!-- confidence:high ✓ -->

1. Masuk ke akun Mekari Qontak CRM dan buka menu **Task**. Sistem akan menampilkan daftar seluruh Task Anda.

2. Untuk mengunduh Task tertentu: Centang checkbox pada Task yang ingin diunduh, kemudian klik tombol **Download** dan pilih **Download Selected**. Sistem akan memproses permintaan pengunduhan.

3. Untuk mengunduh semua Task: Klik tombol **Download** dan pilih **Download All**. Sistem akan memproses seluruh data Task.

4. Cek progress pengunduhan: Buka menu **Properties**, kemudian pilih tab **Upload/Download**. Sistem akan menampilkan status dan progress pengunduhan data Task Anda.

## Expected Result  <!-- confidence:high ✓ -->

Setelah mengikuti langkah-langkah di atas, Anda akan melihat:

• Notifikasi sistem yang mengindikasikan pengunduhan data sedang diproses
• File Task dalam format yang dapat diunduh ke perangkat Anda
• Progress status di menu Properties tab Upload/Download menunjukkan persentase penyelesaian
• File berhasil tersimpan di folder Downloads perangkat Anda setelah proses selesai

Data Task yang diunduh mencakup semua informasi termasuk kolom Reporter yang mengidentifikasi pengguna pembuat atau pengunggah Task.

## Error States  <!-- confidence:low ? -->

No common errors documented.

## Escalation  <!-- confidence:medium ~ -->

Jika Anda mengalami masalah saat mengunduh data Task, hubungi tim support Mekari Qontak dengan informasi berikut:

• Screenshot tampilan error atau pesan notifikasi
• ID akun Mekari Qontak CRM Anda
• Jumlah Task yang mencoba diunduh
• Metode yang digunakan (Download Selected atau Download All)
• Waktu dan tanggal saat error terjadi
• Browser dan versi yang Anda gunakan

Tim support akan membantu mengidentifikasi dan mengatasi masalah pengunduhan data Task Anda.
