## Prerequisites  <!-- confidence:high ✓ -->

• Akun Qontak CRM aktif dan sudah masuk login
• Akses level Member atau Admin
• Akses ke menu Properties di sidebar kiri dashboard
• File logo atau foto profil dalam format dengan background transparan (jika ingin mengunggah gambar)

## Steps  <!-- confidence:high ✓ -->

1. Buka dashboard Qontak CRM Anda
2. Klik menu **Properties** di sidebar kiri
3. Sistem akan menampilkan halaman Properties dengan beberapa pilihan menu
4. Klik tab **Display**
5. Sistem akan menampilkan pengaturan Display dengan opsi unggah logo dan pilihan warna tema
6. (Opsional) Klik tombol **Choose File** untuk mengunggah logo perusahaan atau foto profil Anda
7. (Opsional) Pilih warna tema dashboard dengan mengklik warna yang Anda inginkan, lalu beri centang pada checkbox warna tema tersebut
8. Klik tombol **Save Changes** untuk menyimpan semua kustomisasi tampilan

## Expected Result  <!-- confidence:high ✓ -->

Setelah mengikuti langkah-langkah di atas, halaman Display akan menampilkan konfirmasi bahwa perubahan telah disimpan. Tampilan dashboard CRM Anda akan diperbarui sesuai logo dan warna tema yang Anda pilih. Perubahan hanya berlaku untuk akun Anda sendiri dan tidak memengaruhi tampilan pengguna lain.

## Error States  <!-- confidence:medium ~ -->

• Jika file tidak terunggah: Pastikan file logo atau foto profil memiliki background transparan dan ukuran file tidak terlalu besar
• Jika tombol Save Changes tidak merespons: Pastikan Anda telah memilih setidaknya satu elemen kustomisasi (logo atau warna tema) sebelum menyimpan
• Jika perubahan warna tidak terlihat: Refresh halaman browser Anda dengan menekan Ctrl+F5 (atau Cmd+Shift+R di Mac)

## Escalation  <!-- confidence:medium ~ -->

Jika perubahan Display tidak tersimpan setelah mengikuti semua langkah, atau jika tombol Save Changes tidak berfungsi sama sekali, hubungi tim support Qontak dengan menyertakan:
• Screenshot halaman Display yang mengalami masalah
• ID akun atau email terkait
• Browser dan versi yang Anda gunakan
• Deskripsi langkah yang sudah dicoba
