## Prerequisites  <!-- confidence:high ✓ -->

Untuk mengedit rincian pengguna (User Details) di Qontak CRM, Anda memerlukan:

• Akses level Admin di Qontak CRM perusahaan Anda (jika ingin mengubah data user lain)
• Akses ke dashboard CRM Qontak
• Jika Anda bukan Admin, Anda hanya dapat mengedit profil akun Anda sendiri
• Login aktif ke sistem Qontak CRM

## Steps  <!-- confidence:high ✓ -->

1. Klik tanda panah di sebelah kanan username Anda (kanan atas dashboard CRM). Menu dropdown akan muncul.

2. Pilih "Profile Settings" dari menu dropdown. Anda akan melihat daftar nama-nama pengguna yang terdaftar di CRM perusahaan Anda.

3. Pilih akun pengguna yang ingin Anda ubah dari daftar tersebut.

4. Pada halaman User Details, ubah informasi pengguna sesuai kebutuhan (nama, nomor telepon, NIK, Staff Level, Role, dan pengaturan lainnya).

5. Jika diperlukan, ubah pengaturan tambahan seperti User Performance (target penjualan), User Permissions (pembatasan akses database), atau Tracking Time (jam kerja dan absensi).

6. Klik tombol "Update User" untuk menyimpan semua perubahan yang telah Anda lakukan. Sistem akan menyimpan data pengaturan terbaru.## Expected Result  <!-- confidence:high ✓ -->

Setelah mengklik "Update User", sistem berhasil menyimpan semua perubahan rincian pengguna. Anda akan tetap berada di halaman User Details atau kembali ke daftar pengguna dengan data terbaru yang sudah terupdate. Perubahan yang Anda lakukan akan langsung berlaku untuk akun pengguna tersebut.

## Error States  <!-- confidence:high ✓ -->

Beberapa pengaturan tidak dapat diubah melalui fitur Edit User Details, di antaranya:

• Password: Tidak dapat diubah langsung di Edit User Details
• Email Address: Tidak dapat diubah langsung di Edit User Details

Jika Anda perlu mengubah Password atau Email Address, hubungi tim Support atau konsultan Qontak untuk membantu perubahan tersebut, karena memerlukan bantuan tim kami.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim Support atau konsultan Qontak jika mengalami masalah dalam kasus berikut:

• Ingin mengubah Password pengguna
• Ingin mengubah Email Address pengguna
• Mengalami error saat mengklik tombol "Update User"
• Tidak memiliki akses menu "Profile Settings" meskipun sudah login sebagai Admin

Siapkan informasi: nama pengguna yang diubah, deskripsi masalah, dan screenshot jika diperlukan.
