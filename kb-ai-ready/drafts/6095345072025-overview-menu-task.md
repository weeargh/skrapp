## Definition  <!-- confidence:high ✓ -->


Menu Task merupakan menu yang dapat Anda gunakan untuk melakukan manajemen tugas tim atau individu. Di sini Anda dapat mengelola seluruh aktivitas, tugas pengguna atau tugas yang ditunjukan untuk pengguna lain.
Pada bagian ini Anda dapat melihat seluruh daftar Task yang tersimpan pada CRM Qontak.
  1. Berikut adalah tampilan dari Menu **Task.**  
![task1.jpg](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F36772391290777)
  2. Menu ini juga dapat dilihat dalam bentuk calendar dengan klik ikon **"Calendar View"** seperti ini.  
![calendar_view.jpg](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F36772391297433)
  3. Anda juga dapat set per status dengan klik ikon**"View"** seperti ini.  
![per_status.jpg](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F36772398851353)
  4. Selain itu, Anda dapat **menambahkan, mengunduh, mengubah** dan**menghapus** Task secara massal dan Anda dapat juga menemukan Task tertentu dengan melakukan filter pada kolom pencarian.   
![task_edit.jpg](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F36772391291673)
  5. Untuk melihat detail Task, anda dapat klik **Nama Task** yang Anda tuju. Di sini Anda dapat melihat seluruh informasi dan riwayat task Anda, seperti di bawah ini.  
![25.png](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F56956762496537)

## Key Attributes  <!-- confidence:high ✓ -->

• Tampilan daftar dan kalender Task
• Filter berdasarkan status Task dengan ikon View
• Pencarian dan filter kolom untuk menemukan Task tertentu
• Operasi massal: tambah, ubah, hapus, dan unduh Task
• Detail Task lengkap termasuk informasi dan riwayat task
• Kompatibel dengan pengaturan Help Text pada Task properties
• Fitur Freeze Task untuk mengatur kondisi Task yang tidak dapat diubah
• Kemampuan impor Task secara massal melalui file Excel
