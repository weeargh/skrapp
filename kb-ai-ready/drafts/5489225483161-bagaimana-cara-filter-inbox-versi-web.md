## Prerequisites  <!-- confidence:high ✓ -->

- Akun Qontak Omnichannel aktif dengan akses ke menu Inbox
- Role Agent atau Admin untuk mengakses fitur filter
- Minimal satu room chat atau percakapan tersedia di Inbox
- Koneksi internet stabil
- Channel (WhatsApp, Instagram, atau channel lainnya) sudah terintegrasi dengan Qontak Omnichannel
- (Opsional) Agents dan Tags sudah dikonfigurasi pada akun Anda jika ingin memfilter berdasarkan kriteria tersebut

## Steps  <!-- confidence:high ✓ -->

1. Login ke akun Qontak Omnichannel Anda. Dashboard utama akan ditampilkan.
2. Klik menu **Inbox**. Sistem akan menampilkan daftar pesan dan room chat yang tersedia.
3. Klik ikon **Filter** yang terletak di area filter. Panel pengaturan filter akan muncul.
4. Pilih kriteria filter sesuai kebutuhan:
   - **Sessions**: Pilih Open, Expiring, atau Expired
   - **Agents**: Pilih agent dari dropdown
   - **Channels**: Pilih satu atau lebih channel dari dropdown
   - **Tags**: Pilih tag dari dropdown
   - **Whatsapp Campaign**: Centang untuk menampilkan pesan broadcast WhatsApp
   - **Unresponded Chat**: Centang untuk menampilkan pesan yang belum dibalas agent
5. Klik tombol **Apply** atau **Terapkan**. Sistem akan memfilter pesan berdasarkan kriteria yang dipilih dan menampilkan hasilnya di daftar inbox.

## Expected Result  <!-- confidence:high ✓ -->

Daftar pesan di menu Inbox berhasil difilter sesuai kriteria yang Anda tentukan. Hanya pesan yang memenuhi semua filter aktif yang ditampilkan. Anda dapat melihat jumlah pesan yang sesuai filter dan mengklik salah satu untuk membuka detail percakapan. Filter tetap aktif hingga Anda mengubah atau menghapus kriteria filter.

## Error States  <!-- confidence:low ? -->

No common errors documented.

## Escalation  <!-- confidence:medium ~ -->

Jika fitur filter tidak berfungsi atau kriteria filter tidak tersimpan:
1. Refresh halaman browser Anda dan coba kembali.
2. Pastikan koneksi internet Anda stabil.
3. Verifikasi bahwa Anda memiliki role Agent atau Admin di akun Omnichannel.
4. Jika masalah persisten, hubungi support-qontak@mekari.com dengan informasi: nama akun, screenshot masalah, dan daftar filter yang Anda coba gunakan.
