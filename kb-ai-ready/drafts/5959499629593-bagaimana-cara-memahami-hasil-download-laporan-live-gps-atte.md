## Prerequisites  <!-- confidence:medium ~ -->

Untuk memahami hasil download laporan Live GPS Attendance Tracking pada Monthly Report, Anda memerlukan:

1. Akun Qontak CRM aktif dengan peran Admin atau Manager
2. Fitur Live GPS Attendance Tracking sudah ditambahkan pada dashboard
3. Akses ke menu laporan Daily Check-in Report
4. Aplikasi spreadsheet (Microsoft Excel, Google Sheets, atau sejenisnya) untuk membuka file yang diunduh
5. Pemahaman tentang format data kolom laporan GPS

## Steps  <!-- confidence:high ✓ -->

1. Buka dashboard Qontak CRM dan pastikan widget Live GPS Attendance Tracking sudah ditambahkan.
   → Widget akan ditampilkan pada dashboard utama.

2. Klik ikon garis tiga (menu) pada widget Live GPS Attendance Tracking.
   → Menu dropdown akan muncul dengan pilihan laporan.

3. Pilih opsi "Daily Check-in Report".
   → Sistem akan menampilkan halaman laporan dengan data GPS attendance.

4. Klik tombol Download untuk mengunduh laporan dalam format Monthly Report.
   → File spreadsheet akan terunduh ke perangkat Anda.

5. Buka file yang telah diunduh menggunakan aplikasi spreadsheet.
   → Sheet Monthly Report akan menampilkan data attendance detail.## Expected Result  <!-- confidence:high ✓ -->

Setelah berhasil mengunduh dan membuka laporan Live GPS Attendance Tracking Monthly Report, Anda akan melihat spreadsheet dengan kolom-kolom berikut:

- Nama setiap user
- Tanggal check-in dan check-out
- Waktu check-in dan check-out
- Lokasi check-in dan check-out
- Status Present, Absent, Arrived Late, Left Early
- Data Mock Locations (First Check-in dan Last Check-Out)
- Durasi Tardy dalam menit

Setiap baris mewakili satu catatan attendance user dengan filter waktu laporan yang sesuai periode Monthly Report.

## Error States  <!-- confidence:medium ~ -->

Situasi yang mungkin dihadapi:

1. **Widget Live GPS Attendance Tracking tidak terlihat di dashboard**: Pastikan Anda sudah menambahkan widget tersebut melalui pengaturan dashboard. Hubungi admin jika tidak memiliki akses untuk menambah widget.

2. **File tidak terunduh**: Periksa koneksi internet dan izin download di browser Anda. Coba unduh kembali laporan.

3. **File tidak dapat dibuka**: Pastikan Anda menggunakan aplikasi spreadsheet yang kompatibel (Excel, Google Sheets, atau LibreOffice).

4. **Data laporan kosong**: Verifikasi bahwa ada data attendance dalam periode yang dilaporkan.

## Escalation  <!-- confidence:medium ~ -->

Hubungi Qontak Support jika mengalami:

1. **Widget Live GPS Attendance Tracking tidak muncul** meskipun sudah ditambahkan: Sertakan screenshot dashboard Anda.

2. **Laporan tidak dapat diunduh** atau muncul pesan error: Catat pesan error lengkap dan timestamp saat error terjadi.

3. **Data pada laporan tidak sesuai atau tidak lengkap**: Siapkan informasi user yang bermasalah, periode laporan, dan screenshot data dari aplikasi mobile.

4. **File terunduh rusak atau tidak terbaca**: Bagikan file tersebut jika memungkinkan.

Sertakan ID akun Qontak Anda saat menghubungi support.
