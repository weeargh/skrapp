## Prerequisites  <!-- confidence:medium ~ -->

Untuk memahami hasil download Laporan Top Company di Qontak CRM, Anda membutuhkan:

• Akun Qontak CRM aktif dengan kredensial login
• Peran pengguna: Admin, Owner, atau Agen
• Akses ke menu Report di Qontak CRM
• Laporan Top Company sudah diunduh sebelumnya dalam format file
• Aplikasi spreadsheet (seperti Microsoft Excel atau Google Sheets) untuk membuka file hasil download

Catatan: Laporan Top Company hanya menampilkan data penjualan yang telah tercatat di pipeline CRM Anda.

## Steps  <!-- confidence:high ✓ -->


Laporan ini berfungsi untuk melihat kepada perusahaan mana saja produk atau jasa Anda lebih banyak terjual. Anda dapat melihat seberapa besar penjualan pada tiap perusahaan dan berapa persen penjualan tersebut berkontribusi pada keseluruhan total penjualan.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim support Qontak jika mengalami:

• File laporan tidak dapat diunduh atau file rusak saat dibuka
• Data pada laporan tidak sesuai dengan transaksi yang ada di CRM
• Kolom laporan tidak lengkap atau ada data yang hilang
• Filter yang diterapkan (pipeline, user, waktu) tidak berfungsi dengan baik

Sediakan informasi berikut saat menghubungi support:
- Screenshot file laporan atau pesan error
- ID akun Qontak CRM Anda
- Rentang waktu laporan yang diunduh
- Daftar perusahaan/transaksi yang seharusnya muncul di laporan
