## Prerequisites  <!-- confidence:high ✓ -->

Untuk memahami dan mengatasi Batas Template Pesan Marketing untuk Setiap Pengguna (Per-User Marketing Template Message Limits), Anda memerlukan:

• Akses ke fitur WhatsApp Broadcast di Qontak Omnichannel atau Qontak Chat
• Akses ke menu Campaign (Broadcast) dan Broadcast Report
• Template kategori Marketing yang sudah dibuat dan aktif
• Pemahaman dasar tentang error code 131049 dari Meta
• Akses ke dokumentasi Meta terkait Marketing Template Message Limits

## Steps  <!-- confidence:high ✓ -->

**Untuk Memantau dan Merespons Batas Template Pesan Marketing:**

1. Buka menu Campaign (Broadcast) dan pilih Broadcast Report untuk melihat status pesan yang dikirim.
2. Identifikasi pesan dengan status "Failed" atau error code 131049 dengan pesan "This message was not delivered to maintain healthy ecosystem engagement".
3. Analisis kinerja template Anda menggunakan fasilitas Broadcast Report untuk melihat tingkat pembukaan dan balasan.
4. Optimalkan konten template dengan fokus pada kualitas daripada kuantitas pesan.
5. Gunakan template Follow-up untuk membangun percakapan dua arah dengan penerima pesan.

## Expected Result  <!-- confidence:high ✓ -->

Setelah mengikuti langkah-langkah di atas, Anda akan:

• Dapat mengidentifikasi pesan mana yang terdampak Frequency Capping Meta
• Memahami bahwa error code 131049 menunjukkan pembatasan Meta terhadap pengguna tertentu
• Melihat laporan detail di Broadcast Report yang menampilkan tingkat keterlibatan pelanggan
• Dapat menyesuaikan strategi pemasaran dengan mengirim pesan yang lebih berkualitas dan personal
• Meningkatkan engagement rate melalui percakapan bermakna, bukan promosi sepihak

## Error States  <!-- confidence:high ✓ -->

**Error Code 131049 – Pesan Tidak Terkirim untuk Menjaga Ekosistem yang Sehat**

Error ini muncul di Broadcast Log ketika:
• Pengguna penerima tidak menunjukkan engagement tinggi (tidak membuka atau membalas pesan sebelumnya)
• Meta mendeteksi pola pengiriman yang agresif atau kurang relevan
• Batas dinamis (dynamic capping) Meta untuk pengguna tersebut telah tercapai

**Cara mengatasi:**
Tingkatkan kualitas template, personalisasi konten, ajak pelanggan dalam percakapan dua arah, dan pantau Broadcast Report secara berkala untuk melihat respons penerima.

## Escalation  <!-- confidence:medium ~ -->

**Hubungi Tim Dukungan Qontak jika:**

• Error code 131049 terus muncul meskipun Anda sudah mengoptimalkan kualitas template
• Semua pesan broadcast ke pengguna tertentu gagal terkirim tanpa alasan yang jelas
• Anda ingin mendapatkan analisis mendalam tentang kinerja template dari tim support

**Informasi yang perlu disiapkan:**
• Screenshot Broadcast Log dengan error code 131049
• ID template yang bermasalah
• Jumlah pengguna yang terdampak
• Riwayat engagement pengguna (dari Broadcast Report)
• Tanggal dan waktu pengiriman pesan yang gagal
