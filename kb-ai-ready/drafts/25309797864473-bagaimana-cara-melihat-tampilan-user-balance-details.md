## Prerequisites  <!-- confidence:high ✓ -->

Untuk melihat tampilan User Balance Details di Qontak Omnichannel, Anda membutuhkan:

- Akun Qontak Omnichannel aktif dengan akses Admin atau Supervisor
- Akses ke menu Package Usage
- Paket berlangganan aktif (Qontak Omnichannel atau Qontak Chat)
- Koneksi internet stabil untuk membuka Web Qontak Omnichannel
- Setidaknya satu paket conversation (WhatsApp atau channel lainnya) yang sudah aktif

## Steps  <!-- confidence:high ✓ -->

1. Login ke akun Qontak Omnichannel Anda menggunakan kredensial Admin atau Supervisor.

2. Di menu utama, pilih **Package Usage**, kemudian klik tab **Conversation**. Sistem akan menampilkan halaman Overview Package Usage dengan informasi saldo pengguna.

3. Klik tombol **View details** pada bagian Balance. Sistem akan menampilkan halaman detail saldo yang menunjukkan tiga kategori informasi: Initial balance, Additional balance, dan Total remaining balance.

4. Lihat rincian setiap kategori saldo untuk memantau perkembangan penggunaan paket Anda.

## Expected Result  <!-- confidence:high ✓ -->

Halaman User Balance Details menampilkan tiga kategori informasi:

- **Initial balance**: Saldo awal paket (angka kiri = sisa saldo, angka kanan = total awal)
- **Additional balance**: Saldo dari add-ons yang dibeli (angka kiri = sisa saldo, angka kanan = total awal)
- **Total remaining balance**: Jumlah total sisa saldo keseluruhan

Informasi ini membantu Anda memantau penggunaan quota WhatsApp Conversation dan Monthly Unique Visitor secara rinci.

## Error States  <!-- confidence:medium ~ -->

No common errors documented.

## Escalation  <!-- confidence:medium ~ -->

Jika Anda mengalami kesulitan mengakses menu Package Usage atau tampilan User Balance Details tidak muncul dengan benar:

1. Pastikan koneksi internet stabil dan refresh halaman browser.
2. Verifikasi bahwa akun Anda memiliki akses Admin atau Supervisor.
3. Jika masalah berlanjut, hubungi Qontak Support melalui email support-qontak@mekari.com dengan menyertakan screenshot halaman yang bermasalah dan ID akun Anda.
