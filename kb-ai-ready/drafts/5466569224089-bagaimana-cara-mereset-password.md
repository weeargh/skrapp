## Prerequisites  <!-- confidence:high ✓ -->

Untuk mereset password akun Qontak CRM Anda, Anda memerlukan:
• Akun Qontak CRM yang aktif dan terdaftar
• Akses email yang sama dengan email terdaftar di akun CRM Anda (untuk menerima link reset password)
• Untuk reset password pengguna lain: peran Admin pada organisasi CRM Anda
• Koneksi internet yang stabil
• Aplikasi Qontak versi Web (reset password hanya tersedia di versi Web, bukan mobile)

## Steps  <!-- confidence:high ✓ -->

**Opsi A: Dari Halaman Login**
1. Buka halaman login Qontak CRM.
2. Klik tombol "Forgot Password" pada halaman login.
3. Masukkan username akun Anda, kemudian klik "Send me reset password instructions".
4. Sistem akan mengirimkan link reset password ke alamat email terdaftar Anda.
5. Buka email dan klik link reset password untuk mengatur kata sandi baru.
6. Login kembali dengan password baru Anda.

**Opsi B: Dari Menu Profile Settings**
1. Login ke akun Qontak CRM Anda.
2. Klik panah di sebelah kanan username Anda (kanan atas dashboard).
3. Pilih "Profile Settings".
4. Klik tab "Profile Settings" di sisi kiri halaman.
5. Isi kolom "Existing Password", "New Password", dan "Confirm New Password".
6. Klik tombol "Update". Sistem akan menampilkan konfirmasi perubahan password berhasil.

**Opsi C: Reset Password Pengguna Lain (Admin Only)**
1. Login sebagai Admin.
2. Klik panah di sebelah kanan username Anda (kanan atas dashboard).
3. Pilih "Profile Settings".
4. Sistem akan menampilkan daftar semua user terdaftar di organisasi CRM.
5. Temukan user yang ingin di-reset passwordnya.
6. Klik tombol "Reset Password" di sebelah kanan nama user tersebut.
7. Sistem akan mengirimkan link reset password ke email user tersebut.

## Expected Result  <!-- confidence:high ✓ -->

Setelah berhasil mereset password:
• Halaman login menampilkan pesan konfirmasi reset password berhasil
• Anda dapat login menggunakan password baru Anda
• Akun Anda tetap aktif dengan semua data dan pengaturan sebelumnya terjaga
• Untuk reset password pengguna lain: user tersebut menerima email dengan link reset password dan dapat mengatur password baru mereka sendiri

## Error States  <!-- confidence:high ✓ -->

**Email tidak menerima link reset password:**
• Periksa folder Spam atau Junk di email Anda
• Pastikan email yang terdaftar di CRM benar-benar aktif dan dapat menerima email
• Tunggu beberapa menit sebelum mencoba kembali

**Tidak ingat username akun:**
• Hubungi Admin organisasi Anda untuk mengetahui username Anda

**Link reset password expired atau tidak berfungsi:**
• Ulangi proses reset password dari awal untuk mendapatkan link baru

Jika masalah persisten terkait email, hubungi tim Support Qontak untuk bantuan lebih lanjut.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim Support Qontak atau konsultan Anda jika:
• Email berisi link reset password tidak diterima setelah menunggu 15 menit
• Link reset password tidak berfungsi meskipun sudah dicoba berkali-kali
• Fitur "Forgot Password" atau "Reset Password" tidak muncul atau tidak responsif
• Tidak dapat mengakses aplikasi Qontak versi Web

Sediakan informasi berikut saat menghubungi support: username akun, email terdaftar, screenshot error (jika ada), dan langkah-langkah yang sudah Anda coba.
