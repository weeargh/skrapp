## Prerequisites  <!-- confidence:high ✓ -->

Untuk mengelola menu Properties - Products di Mekari Qontak, Anda membutuhkan:

- Akun Mekari Qontak CRM aktif dengan modul Products tersedia
- Akses ke menu Properties dengan izin admin atau manajemen
- Login berhasil ke Qontak versi Web
- Pemahaman tentang jenis-jenis field type yang ingin ditambahkan pada properti produk

## Steps  <!-- confidence:high ✓ -->

**Menambahkan Properti Baru:**

1. Buka menu Properties dan pilih tab Product. Sistem menampilkan daftar properti yang ada.
2. Klik tombol "Buat Properti Baru". Formulir tambah properti akan muncul.
3. Isi field Name (nama properti) sesuai kebutuhan Anda.
4. Pilih Field Type dari dropdown (Single Line Text, Dropdown Select, Number, Date, Upload, Multiple Select, Percentage, Text Area, Photo, URL, Signature, atau GPS).
5. (Opsional) Centang checkbox Required jika properti wajib diisi.
6. (Opsional) Centang checkbox Show untuk menampilkan di detail produk.
7. (Opsional) Centang checkbox Show in List untuk menampilkan di daftar produk.
8. Pilih Styling (Short Column atau Long Column).
9. Klik tombol "Save". Properti baru berhasil ditambahkan.

**Mengedit Properti:**

1. Di menu Properties → Product, cari properti yang ingin diubah.
2. Klik tombol "Edit". Form edit properti akan muncul.
3. Ubah label, nama, field type, required, show, show in List, atau styling sesuai kebutuhan.
4. Klik tombol "Save". Perubahan berhasil disimpan.

**Menghapus Properti:**

1. Di menu Properties → Product, cari properti yang ingin dihapus.
2. Klik tombol "Destroy". Properti akan dihapus dari sistem.

Catatan: Default field tidak dapat dihapus dan tidak menampilkan tombol "Destroy".## Expected Result  <!-- confidence:high ✓ -->

Setelah menambahkan properti baru:
- Properti muncul di daftar menu Properties → Product
- Field baru tersedia saat membuat atau mengedit produk di menu Produk
- Jika checkbox Show diaktifkan, properti tampil di halaman detail produk
- Jika checkbox Show in List diaktifkan, properti tampil sebagai kolom dalam daftar produk
- Jika checkbox Required diaktifkan, produk tidak dapat disimpan tanpa mengisi properti tersebut

Setelah mengedit atau menghapus properti, perubahan langsung berlaku untuk semua produk yang menggunakan properti tersebut.

## Error States  <!-- confidence:medium ~ -->

**Field Name tidak diisi:**
Sistem akan menampilkan pesan validasi. Isi field Name dengan nama properti sebelum menyimpan.

**Field Type tidak dipilih:**
Sistem akan menampilkan pesan validasi. Pilih salah satu tipe field dari dropdown Field Type sebelum menyimpan.

**Properti default tidak dapat dihapus:**
Beberapa properti bawaan (default field) seperti Product Name tidak memiliki tombol "Destroy". Properti ini tidak dapat dihapus; Anda hanya dapat menyembunyikannya dengan unchecking checkbox Show.

**Perubahan tidak tersimpan:**
Pastikan Anda mengklik tombol "Save" setelah mengisi atau mengubah data properti.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim support Mekari Qontak jika Anda mengalami:

- Tidak dapat mengakses menu Properties meskipun memiliki izin admin/manajemen
- Tombol "Save" tidak merespons atau properti tidak tersimpan setelah mengklik "Save"
- Field type tertentu tidak tersedia di dropdown Field Type
- Properti yang sudah dihapus masih muncul di menu Produk
- Error message yang tidak jelas saat menambah atau mengedit properti

Sediakan informasi: akun email Qontak, screenshot error, browser yang digunakan, dan langkah-langkah yang sudah dicoba sebelum menghubungi support.
