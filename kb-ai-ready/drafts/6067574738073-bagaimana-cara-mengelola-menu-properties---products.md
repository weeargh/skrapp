## Prerequisites  <!-- confidence:high ✓ -->

Untuk mengelola menu Properties - Products di Mekari Qontak, Anda membutuhkan:

- Akun Mekari Qontak CRM aktif dengan modul Products tersedia
- Akses ke menu Properties dengan izin admin atau manajemen
- Login berhasil ke Qontak versi Web
- Pemahaman tentang jenis-jenis field type yang ingin ditambahkan pada properti produk

## Steps  <!-- confidence:high ✓ -->


**Properties - Product** adalah cara untuk menambahkan properti pada menu **Produk** atau mengedit properti yang sudah ada pada menu Products. Untuk mengedit dan menambahkan field/properti pada Menu **Product** maka Anda perlu melakukan beberapa langkah sebagai berikut:
  1. Pilih Menu **Properties**.
  2. Pilih tab **Product**. Klik**“Buat Properti Baru”** untuk menambahkan field/properti pada Menu Product.  
![39.png](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F53836354387353)
  3. Kemudian, akan muncul tampilan seperti berikut. Selanjutnya, Anda dapat mengisi keterangan dari properti tersebut. Lalu, klik **“Save”** untuk menyimpan properti baru.  
![2.png](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F36780919355673)
Berikut adalah keterangan field yang dapat Anda isi.  
**1. Name (wajib diisi)** : Name merupakan nama dari properti baru yang ingin kita buat pada menu **Product.**  
**2.** **Field Type (wajib diisi)** : Field type berisi pilihan dari tipe pengisian properti yang Anda inginkan. Apabila Anda mengklik field type, maka akan muncul beberapa pilihan tipe properti yang dapat Anda pilih, antara lain:  
- **Single Line Text 🡪** pengisian field dengan teks singkat.  
- **Dropdown Select 🡪** memungkinkan user untuk memilih satu dari beberapa opsi yang muncul sebagai isi field tersebut.  
- **Number 🡪** field hanya dapat diisi dengan angka.  
- **Date 🡪** field hanya dapat diisi dengan tanggal dan user langsung dapat mengisi secara manual dengan format dd/mm/yyyy atau mengklik dari kalendar yang muncul pada saat pengisian.  
- **Upload 🡪** pengisian field dilakukan dengan mengunggah file.  
- **Multiple Select 🡪** memungkinkan user untuk memilih lebih dari satu opsi yang muncul sebagai isi field tersebut.  
- **Percentage 🡪** isi field akan otomatis terkonversi menjadi persentase.  
- **Text Area 🡪** pengisian field dengan teks yang panjang.  
- **Photo 🡪** CRM akan secara otomatis akan membuka kamera untuk pengambilan gambar secara live. User tidak dapat mengunggah gambar dari galeri.  
- **URL 🡪** pengisian field dengan link.  
- **Signature 🡪** field diisi dengan tanda tangan secara langsung.  
- **GPS 🡪** CRM akan mendeteksi lokasi GPS pada pengisian field dengan tipe GPS dilakukan  
**3. Required:** Apabila checkbox required diklik hingga menunjukkan tanda centang, maka properti/field tersebut nantinya wajib diisi oleh user, sehingga kontak tidak dapat disimpan bila properti/field tersebut tidak diisi  
**4. Show:** Apabila checkbox show diklik hingga menunjukkan tanda centang, maka keterangan properti/field tersebut akan muncul detail kontak pada Menu **Product**  
**5. Show in List:** Centang disini Apabila Anda ingin keterangan properti/kolom tersebut akan muncul pada list produk  
**6. Styling:** berfungsi untuk memilih style short column atau long column untuk kolom konten
  4. Untuk mengedit field/properti, Anda bisa klik **“Edit”**.  
![3.png](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F36780961485209)
  5. Melalui fitur edit, Anda dapat mengubah label, nama, field type, required, show, show in List, dan Styling . Apabila perubahan sudah selesai diatur, Anda dapat klik**“Save”**.  
![5.png](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F36780961486745)
  6. Untuk menghapus properti/field yang sudah dibuat Anda klik **“Destroy”.**  
**![4.png](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F36780919357977)**

Terdapat beberapa default field yang tidak dapat dihapus dan tidak memberikan opsi “**Destroy** ”, maka apabila Anda ingin menghilangkan properti/field tersebut dari detail kontak, Anda harus mengedit properti/field tersebut dan mengosongkan checkbox show. Kemudian, klik tombol “**Save** ”.
  7. Kemudian, akan muncul popup **“Are you sure to removed field”,** kemudian pilih Remove untuk menghapus field yang diinginkan.  
![6.png](/mekarirag/proxy/image?url=https%3A%2F%2Fhelp-center.qontak.com%2Fhc%2Farticle_attachments%2F36780919359385)

## Error States  <!-- confidence:medium ~ -->

**Field Name tidak diisi:**
Sistem akan menampilkan pesan validasi. Isi field Name dengan nama properti sebelum menyimpan.

**Field Type tidak dipilih:**
Sistem akan menampilkan pesan validasi. Pilih salah satu tipe field dari dropdown Field Type sebelum menyimpan.

**Properti default tidak dapat dihapus:**
Beberapa properti bawaan (default field) seperti Product Name tidak memiliki tombol "Destroy". Properti ini tidak dapat dihapus; Anda hanya dapat menyembunyikannya dengan unchecking checkbox Show.

**Perubahan tidak tersimpan:**
Pastikan Anda mengklik tombol "Save" setelah mengisi atau mengubah data properti.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim support Mekari Qontak jika Anda mengalami:

- Tidak dapat mengakses menu Properties meskipun memiliki izin admin/manajemen
- Tombol "Save" tidak merespons atau properti tidak tersimpan setelah mengklik "Save"
- Field type tertentu tidak tersedia di dropdown Field Type
- Properti yang sudah dihapus masih muncul di menu Produk
- Error message yang tidak jelas saat menambah atau mengedit properti

Sediakan informasi: akun email Qontak, screenshot error, browser yang digunakan, dan langkah-langkah yang sudah dicoba sebelum menghubungi support.
