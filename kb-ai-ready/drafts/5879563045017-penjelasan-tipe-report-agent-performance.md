## Definition  <!-- confidence:high ✓ -->

Agent Performance adalah tipe laporan di Qontak Omnichannel yang menampilkan metrik kinerja agen secara komprehensif. Laporan ini dapat diakses melalui menu Reports dengan memilih tab Agent Performance. Laporan ini menyajikan data tentang waktu respons, waktu resolusi, jumlah percakapan, pesan yang dikirim, serta status ketersediaan agen. Fitur ini dirancang untuk membantu supervisor dan manager memantau produktivitas dan kualitas layanan setiap agen dalam tim.

## Why It Matters  <!-- confidence:medium ~ -->

Agent Performance membantu organisasi memahami efektivitas tim customer service mereka. Dengan laporan ini, supervisor dapat mengidentifikasi agen berprestasi tinggi, mendeteksi area yang memerlukan peningkatan, dan membuat keputusan berdasarkan data tentang pengelolaan tim. Kombinasi metrik seperti First Response Time, Resolution Time, dan CSAT score memberikan gambaran menyeluruh tentang kualitas layanan pelanggan dan efisiensi operasional.

## Key Attributes  <!-- confidence:high ✓ -->

• Metrik utama: First Response Time, Response Time, Resolution Time, Wait Time
• Data percakapan: Assigned Conversations, Resolved Conversations, Messages Sent
• Fitur pemeringkatan: Top Rated Agents by CSAT dan CES (dapat diaktifkan di Settings)
• Perbandingan performa: Agent Performance Comparison untuk membandingkan metrik antar agen
• Status real-time: Agent Status List menampilkan status online/offline dan perubahan status terakhir
• Monitoring: On Going Conversations dan Hand Over Conversation (jika diaktifkan)
• Filter: Dapat disesuaikan berdasarkan zona, channel, dan jangka waktu
