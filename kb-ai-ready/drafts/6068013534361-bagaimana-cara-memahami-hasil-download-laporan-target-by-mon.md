## Prerequisites  <!-- confidence:medium ~ -->

Untuk memahami hasil download Laporan Target By Months By Value di Qontak CRM, Anda memerlukan:

• Akun Qontak CRM aktif dengan kredensial login
• Peran pengguna: Admin, Supervisor, atau Agen
• Akses ke menu Report atau Dashboard di Qontak CRM
• Laporan Target By Months By Value sudah diunduh sebelumnya
• Aplikasi spreadsheet (seperti Microsoft Excel atau Google Sheets) untuk membuka file hasil download

Catatan: File yang diunduh akan berformat spreadsheet dengan beberapa sheet yang menampilkan data target dan pencapaian per user.
