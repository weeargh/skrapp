## Prerequisites  <!-- confidence:high ✓ -->

Untuk mengunggah Recipient List pada fitur Campaign (Broadcast), Anda memerlukan:

• Akun Qontak Omnichannel aktif dengan akses ke menu Campaign
• Role/izin sebagai Admin atau user dengan akses ke fitur Campaign
• Koneksi internet stabil
• File template dalam format yang didukung Qontak (unduh template dari sistem terlebih dahulu)
• Untuk WhatsApp Campaign: maksimal 50.000 recipients per file upload
• Untuk Email Campaign: jumlah recipients sesuai kuota dasar harian email Anda (berlaku sejak 15 September 2025)

## Steps  <!-- confidence:high ✓ -->


Menu **Campaign (Broadcast)** adalah salah satu fitur pada**Chat Panel** yang memungkinkan penggunanya untuk bisa mengirimkan pesan ke banyak orang sekaligus. Fitur tersebut juga biasa dikenal dengan pesan siaran. Melalui Campaign, kita tidak perlu membuka kolom _chat_ tiap-tiap orang di kontak untuk mengirimkan mereka pesan yang sama. Tidak hanya itu, waktu yang dibutuhkan untuk mengirim pesan ke banyak kontak menjadi lebih singkat dan lebih mudah. Pada penjelasan berikut ini, Anda akan mempelajari cara mengunggah 
Sebelumnya Anda dapat mempelajari tentang:  
[**[Blog] Cara Mudah Membuat Pesan Siaran di WhatsApp**](https://qontak.com/blog/cara-membuat-siaran-whatsapp/?utm_source=ecosystem&utm_medium=qontak+%28help+center%29)
Berikut merupakan cara membuat list pesan untuk mengirim Campaign.

  2. Pilih menu**Campaign** dan klik **"Recipient lists"**.

## Error States  <!-- confidence:medium ~ -->

Format file tidak sesuai: Pastikan file template diunduh dari sistem Qontak dan diisi sesuai format yang ditentukan (kolom yang wajib diisi, tipe data, dll).

Jumlah recipients melebihi batas: Untuk WhatsApp, maksimal 50.000 per upload. Untuk Email, sesuaikan dengan kuota dasar harian Anda. Jika melebihi, bagi data ke beberapa file upload.

Data recipients tidak valid: Verifikasi nomor telepon atau email sudah benar dan lengkap sebelum import.

File tidak terbaca: Pastikan file dalam format yang didukung dan tidak corrupt.

## Escalation  <!-- confidence:medium ~ -->

Hubungi tim support Qontak jika Anda mengalami:

• Pop-up notifikasi error yang tidak dapat diperbaiki meskipun sudah mengikuti langkah troubleshooting
• File template tidak dapat diunduh dari sistem
• Masalah akses ke menu Campaign atau Recipient lists
• Konfirmasi kuota email yang tidak sesuai dengan plan Anda

Siapkan informasi berikut saat menghubungi support: screenshot error, ID akun Qontak, nama recipient list, jumlah recipients, dan jenis campaign (WhatsApp/Email).
