## Prerequisites  <!-- confidence:high ✓ -->

Untuk menambahkan Tasks satuan pada aplikasi Qontak CRM mobile, Anda memerlukan:

• Akun Mekari Qontak CRM yang aktif
• Aplikasi Qontak CRM terinstal pada perangkat mobile (iOS atau Android)
• Username dan Password untuk login
• Akses ke menu Task dalam aplikasi mobile
• Koneksi internet yang stabil

Prerequisite ini memastikan Anda dapat mengakses aplikasi dan membuat task baru secara individual.

## Steps  <!-- confidence:high ✓ -->

1. Buka aplikasi **Qontak CRM** pada perangkat mobile Anda.
2. Masukkan Username dan Password, lalu klik tombol **Login**. Anda akan masuk ke dashboard aplikasi.
3. Klik menu **Task**, kemudian klik ikon **+** untuk menambah data Task baru. Formulir Task akan terbuka.
4. Pada bagian **Task Information**, isi semua kolom sesuai data yang diinginkan. Kolom bertanda bintang (*) wajib diisi.
5. Pada bagian **Association**, tentukan asosiasi dengan company, contact (baru atau existing), atau Deal.
6. Pada bagian **Other**, isi Detail/plan, Next step/result, Live GPS location, Attachment, Photo, Signature, dan To-do list Task sesuai kebutuhan.
7. Klik tombol **Save**. Sistem akan menyimpan data Task.
8. Task yang telah dibuat akan muncul pada halaman utama **Task**.

## Expected Result  <!-- confidence:high ✓ -->

Setelah Anda mengklik tombol **Save**, sistem akan menyimpan data Task dan menampilkan konfirmasi penyimpanan berhasil. Anda akan diarahkan kembali ke halaman utama menu **Task**, tempat Task yang baru dibuat akan muncul dalam daftar Tasks dengan semua informasi yang telah Anda masukkan (judul, deskripsi, asosiasi, dan detail lainnya).

## Error States  <!-- confidence:medium ~ -->

No common errors documented.

## Escalation  <!-- confidence:medium ~ -->

Jika Anda mengalami masalah saat menambahkan Tasks satuan pada aplikasi Qontak CRM mobile, hubungi tim support Mekari Qontak dengan informasi berikut:

• ID akun Mekari Qontak Anda
• Versi aplikasi Qontak CRM yang digunakan
• Jenis perangkat mobile (iOS/Android) dan versi sistem operasi
• Screenshot atau deskripsi detail tentang masalah yang dihadapi
• Langkah-langkah yang telah Anda coba untuk mengatasi masalah

Kontak support tersedia melalui help center atau aplikasi.
