## Prerequisites  <!-- confidence:high ✓ -->

Untuk mengelola Package Usage Conversation di Qontak Omnichannel, Anda membutuhkan:

- Akun Qontak Omnichannel aktif dengan akses Admin atau Supervisor
- Paket berlangganan valid (Qontak Omnichannel atau Qontak Chat)
- Akses ke antarmuka web Qontak
- Setidaknya satu paket conversation (WhatsApp atau channel lainnya) yang sudah aktif
- Pengetahuan tentang struktur menu Package Usage di dashboard Qontak

## Steps  <!-- confidence:high ✓ -->

1. Login ke akun Qontak Omnichannel Anda menggunakan kredensial Admin atau Supervisor. Sistem akan menampilkan dashboard utama.

2. Di menu utama, pilih menu Package Usage, kemudian klik tab Conversation. Sistem akan menampilkan halaman Overview Package Usage dengan grafik dan informasi penggunaan quota.

3. Pada bagian atas kanan halaman, Anda akan melihat tiga tombol: Learn More, See purchase history, dan Top up balance. Klik tombol sesuai kebutuhan Anda.

4. Pada bagian Balance, lihat sisa saldo kuota WhatsApp Conversation dan Monthly Unique Visitor yang tersisa untuk bulan berjalan. Sistem menampilkan informasi real-time penggunaan Anda.

5. Pada bagian Total Balance, lihat saldo dari top up yang telah dilakukan untuk Business Initiated messages.

## Expected Result  <!-- confidence:high ✓ -->

Setelah mengikuti langkah-langkah di atas, Anda akan melihat:

- Halaman Package Usage Conversation menampilkan grafik dan report penggunaan quota
- Bagian Balance menampilkan: Available Users, Unique Visitor (initial + top up), dan kuota WhatsApp Conversation yang tersisa
- Bagian Total Balance menampilkan saldo untuk Business Initiated messages
- Informasi reset kuota WhatsApp dan Unique Visitor setiap awal bulan
- Akses ke tombol Learn More, See purchase history, dan Top up balance untuk tindakan lanjutan sesuai kebutuhan Anda

## Error States  <!-- confidence:low ? -->

No common errors documented.

## Escalation  <!-- confidence:high ✓ -->

Jika mengalami masalah saat mengakses halaman Package Usage Conversation atau informasi saldo tidak menampilkan dengan benar, hubungi tim support Qontak dengan menyertakan:

- Email akun Qontak Omnichannel Anda
- Screenshot halaman Package Usage yang mengalami masalah
- Deskripsi masalah yang dihadapi (contoh: saldo tidak ter-update, grafik tidak tampil)
- Waktu masalah terjadi

Hubungi support-qontak@mekari.com atau gunakan fitur Chat support di dashboard Qontak.
