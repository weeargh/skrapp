## Prerequisites  <!-- confidence:high ✓ -->

Untuk menginstall Bizphone Desktop App, Anda memerlukan:

**Untuk Windows:**
- Sistem Operasi Windows 7 SP1 atau versi lebih baru
- .NET Framework 4.5.2 atau versi lebih baru (installer akan mengunduhnya secara otomatis jika belum tersedia)
- Visual Studio 2013 dan 2015 32-bit redistributable libraries (sudah termasuk dalam installer)
- Koneksi internet (opsional—jika tidak tersedia, Anda dapat mengunduh prasyarat secara manual sebelum instalasi)

**Untuk Mac OS:**
- Tidak ada persyaratan sistem khusus
- File Bizphone Desktop App yang sudah diunduh

**Untuk kedua sistem operasi:**
- Lisensi IP Bizphone yang aktif untuk masuk dan menggunakan aplikasi setelah instalasi

## Steps  <!-- confidence:high ✓ -->


**Bizphone Desktop App** adalah aplikasi berguna untuk melakukan dan menerima panggilan melalui komputer. Aplikasi ini merupakan klien softphone yang dapat digunakan pada desktop Windows atau Mac.
Untuk mulai menggunakan BizPhone Desktop App, Anda perlu menginstallnya. Aplikasi ini dapat diinstall di Komputer dengan Sistem Operasi Windows atau Mac dengan persyaratan sistem sebagai berikut:
### 1. Cara Install pada Windows[](https://help-center.qontak.com/hc/id/articles/7969640963609-Bagaimana-Cara-Menginstall-Bizphone-Dekstop-App#h_01KP5S256C5JDZJ9S6VQ2RRYWQ)
**Persyaratan Sistem:**
  * Windows 7 SP1 atau versi lebih baru .NET Framework 4.5.2 atau versi lebih baru (jika Anda belum menginstall installer, maka sistem akan mengunduhnya secara otomatis).
  * Visual Studio 2013 dan 2015 32-bit redistributable libraries (sudah termasuk installer).

Installer akan terunduh dan membuat _library_ yang diperlukan secara otomatis jika belum pernah menginstallnya. Tetapi, jika akses Internet tidak tersedia, Anda dapat mengunduh dan menginstall prasyarat tersebut secara manual sebelum menginstall BizPhone.
Berikut cara menginstall Bizphone pada Windows Anda:
  1. Unduh Bizphone Dekstop App. Pelajari caranya [di sini.](https://help-center.qontak.com/hc/id/articles/7966615586073-Cara-Mengunduh-Bizphone-Dekstop-App)
  2. Setelah file diunduh, ekstrak file .zip untuk menemukan _installer_ aplikasi.

## Error States  <!-- confidence:medium ~ -->

**Error umum dan cara mengatasinya:**

1. **Jendela Installer tidak muncul saat menjalankan file (Windows):** Pastikan Anda menjalankan file dengan "Run as Administrator". Jika masalah berlanjut, unduh dan install .NET Framework 4.5.2 dan Visual Studio 2013/2015 32-bit redistributable libraries secara manual sebelum mencoba instalasi ulang.

2. **Pesan error saat ekstrak file .zip:** File .zip mungkin rusak. Unduh file Bizphone Desktop App kembali dan coba ekstrak ulang.

3. **Aplikasi tidak terbuka setelah instalasi (Windows):** Restart komputer Anda, lalu buka aplikasi Bizphone dari menu Start atau desktop shortcut.

4. **Aplikasi tidak berfungsi setelah instalasi:** Verifikasi bahwa lisensi IP Bizphone Anda aktif dan login dengan kredensial yang benar.

## Escalation  <!-- confidence:medium ~ -->

Hubungi dukungan Qontak jika Anda mengalami:

- Instalasi gagal berulang kali meskipun semua persyaratan sistem sudah terpenuhi
- Pesan error spesifik yang tidak disebutkan di atas muncul saat instalasi
- Aplikasi Bizphone tidak dapat login meskipun lisensi IP Bizphone sudah aktif
- Aplikasi crash atau tidak responsif setelah instalasi berhasil

**Informasi yang perlu disiapkan saat menghubungi support:**
- Sistem operasi dan versi yang Anda gunakan
- Versi .NET Framework dan Visual Studio redistributable yang terinstall (untuk Windows)
- Screenshot pesan error (jika ada)
- Akun email yang terdaftar dengan lisensi Bizphone Anda
