## Prerequisites  <!-- confidence:high ✓ -->

- Akun Qontak Omnichannel aktif
- Aplikasi Qontak Chat terinstal di Smartphone Anda
- Username dan Password Qontak Omnichannel yang valid
- Role Agent atau Admin untuk mengakses fitur filter
- Minimal satu channel yang terintegrasi dengan Omnichannel (WhatsApp, Instagram, atau channel lainnya)
- Koneksi internet stabil di perangkat mobile Anda

## Steps  <!-- confidence:high ✓ -->

1. Buka aplikasi Qontak Chat di Smartphone Anda. Sistem menampilkan halaman login.
2. Masukkan Username dan Password Anda, lalu klik tombol Login. Sistem menampilkan dashboard Omnichannel.
3. Navigasikan ke menu Inbox untuk melihat daftar pesan. Sistem menampilkan list room chat dari semua channel.
4. Tekan ikon filter di bagian atas layar Inbox. Sistem menampilkan panel filter dengan opsi yang dapat dipilih.
5. Pilih filter sesuai kebutuhan Anda:
   - Whatsapp Campaign: untuk pesan broadcast WhatsApp API
   - Tags: untuk pesan dengan tag spesifik
   - Channels: untuk satu atau beberapa channel tertentu
   - Agents: untuk pesan yang ditugaskan ke agent tertentu
6. Klik tombol Apply atau Terapkan. Sistem menampilkan pesan yang sesuai filter pilihan Anda.## Expected Result  <!-- confidence:high ✓ -->

Pesan di menu Inbox berhasil difilter sesuai kriteria yang Anda pilih. Daftar room chat atau pesan akan menampilkan hanya item yang cocok dengan filter Whatsapp Campaign, Tags, Channels, atau Agents yang telah Anda tentukan. Anda dapat melihat jumlah pesan yang tersaring di bagian atas layar dan mulai membalas pesan yang difilter.

## Escalation  <!-- confidence:medium ~ -->

Jika filter tidak berfungsi atau Anda tidak melihat opsi filter di aplikasi mobile:
1. Periksa bahwa versi aplikasi Qontak Chat Anda sudah terbaru — update dari App Store atau Google Play Store jika diperlukan.
2. Logout dan login kembali ke akun Anda.
3. Pastikan role akun Anda adalah Agent atau Admin untuk mengakses fitur filter.
4. Jika masalah berlanjut, hubungi support-qontak@mekari.com dengan informasi: nama akun, screenshot error, dan versi aplikasi Anda.
