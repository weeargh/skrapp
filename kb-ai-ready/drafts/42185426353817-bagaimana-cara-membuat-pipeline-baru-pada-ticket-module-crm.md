## Prerequisites  <!-- confidence:medium ~ -->

Anda ingin membuat Pipeline baru pada modul Ticket CRM untuk mengkategorikan dan mengelola list Ticket yang harus diselesaikan. Sebelum memulai, pastikan:

• Anda memiliki akses ke akun Qontak CRM dengan peran Admin atau User
• Produk Qontak CRM sudah aktif dan terintegrasi
• Anda dapat mengakses menu Properties di dashboard CRM
• Tidak ada batasan jumlah Pipeline yang dapat dibuat

## Steps  <!-- confidence:high ✓ -->

1. Buka akun Qontak CRM Anda, kemudian pilih menu **Properties**. Sistem akan menampilkan halaman Properties dengan beberapa tab.

2. Pilih tab **Tickets**. Sistem menampilkan daftar konfigurasi untuk modul Ticket.

3. Klik tab **Pipeline**, lalu pilih tombol **Create New Pipeline**. Form pembuatan Pipeline baru akan muncul.

4. Isi kolom **Pipeline Name** dengan nama pipeline, tentukan **Nama Stage** dan **Persentase Stages**. Klik **Add New Stages** jika perlu menambah tahapan lainnya.

5. Klik tombol **Create Pipeline**. Sistem akan menyimpan konfigurasi Pipeline baru.

6. Kembali ke menu Ticket untuk melihat Pipeline yang telah dibuat melalui kolom filter.

## Expected Result  <!-- confidence:high ✓ -->

Pipeline baru berhasil dibuat dan muncul dalam daftar Pipeline di Qontak CRM. Sistem menampilkan konfirmasi bahwa Pipeline telah tersimpan. Ketika Anda membuka menu Ticket atau membuat Ticket baru, Pipeline yang baru dibuat akan tersedia sebagai opsi pilihan di kolom filter dan dropdown Pipeline. Setiap Stage yang Anda tentukan akan dapat digunakan untuk mengorganisir Ticket sesuai tahapan pengerjaannya.

## Error States  <!-- confidence:low ? -->

No common errors documented.

## Escalation  <!-- confidence:medium ~ -->

Jika Pipeline tidak muncul setelah dibuat, atau jika Anda mengalami kesulitan mengakses menu Properties atau tab Tickets, hubungi tim support Qontak melalui in-app chat atau email. Siapkan informasi berikut:

• ID akun CRM Anda
• Nama Pipeline yang Anda coba buat
• Screenshot halaman Properties/Pipeline
• Pesan error (jika ada)
